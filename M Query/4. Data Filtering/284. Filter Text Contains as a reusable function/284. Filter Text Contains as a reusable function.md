# Filter Text Contains as a reusable function

**Description:**
This transformation achieves filtering a table to only include rows where a specific text column contains a certain substring. It allows for case-insensitive matching and is useful for data cleansing and data preparation. The function can be reused across different queries and data sources.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when you need to filter data based on a text search. For example, you might want to filter customer feedback data to only include records that mention a specific product or service.
