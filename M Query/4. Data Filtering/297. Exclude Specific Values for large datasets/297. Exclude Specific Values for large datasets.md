# Exclude Specific Values for Large Datasets

**Description:**
This transformation excludes specific values from large datasets using Power Query (M). It filters out unwanted data points, ensuring only relevant information is processed further. This is particularly useful for data cleansing and preparation.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial when dealing with large datasets that contain irrelevant or erroneous data points. For instance, when analyzing customer feedback, you might want to exclude responses that are not in a specific language or contain certain keywords that are not relevant to the analysis.

### Core M Functions Used
`Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"ID", "Name", "Age"},
        {1, "John", 25},
        {2, "Jane", 30},
        {3, "Doe", 20}
    }),
    ExcludedValues = {"John", 30},
    FilteredTable = Table.SelectRows(Source, each not List.Contains(ExcludedValues, [Name]))
in
    FilteredTable
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
1. **Data Source**: Start with a table that contains the data you want to filter.
2. **Specify Excluded Values**: Identify the specific values you want to exclude from your dataset.
3. **Apply Filter**: Use the `Table.SelectRows` function in Power Query (M) to filter out rows that contain the excluded values.
4. **Result**: The resulting table will only include rows that do not match the excluded values, effectively excluding them from further analysis or processing.
