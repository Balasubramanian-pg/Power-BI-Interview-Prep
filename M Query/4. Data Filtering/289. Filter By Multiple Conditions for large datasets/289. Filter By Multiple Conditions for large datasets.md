# Filter By Multiple Conditions for large datasets

**Description:**
This transformation achieves filtering large datasets based on multiple conditions. It allows for efficient data processing by applying various criteria to narrow down the dataset. This is particularly useful in data analysis and reporting where data needs to be refined based on specific conditions.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when analyzing customer data. For instance, a marketing team wants to target customers who are above 18 years old, reside in a specific region, and have made a purchase in the last 30 days. By applying multiple filter conditions, the team can obtain a refined dataset that matches their criteria, enabling more precise marketing strategies.

### Core M Functions Used
`Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"John", 25, "USA", "2022-01-01"},
        {"Jane", 32, "Canada", "2022-02-01"},
        {"Bob", 28, "USA", "2022-03-01"}
    }, {"Name", "Age", "Country", "PurchaseDate"}),
    FilteredRows = Table.SelectRows(Source, each [Age] > 25 and [Country] = "USA" and [PurchaseDate] >= "2022-02-01")
    in
    FilteredRows
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
    1. **Data Source**: The process begins with a data source, which is a table containing customer information such as name, age, country, and purchase date.
    2. **Applying Filters**: The `Table.SelectRows` function is used to apply multiple filter conditions to the data. In this example, the conditions are: `Age > 25`, `Country = "USA"`, and `PurchaseDate >= "2022-02-01"`.
    3. **Filtered Dataset**: The function returns a new table (`FilteredRows`) that includes only the rows from the original table that satisfy all the specified conditions.
    4. **Result**: The resulting dataset is a refined table that contains only the relevant customer data based on the multiple filter conditions applied.
