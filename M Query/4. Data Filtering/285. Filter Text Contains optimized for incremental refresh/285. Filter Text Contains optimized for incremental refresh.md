# Filter Text Contains Optimized for Incremental Refresh

**Description:**
This transformation filters a table to include only rows where a specified text column contains a certain string, optimized for incremental refresh. It allows for efficient data processing by minimizing the amount of data that needs to be re-processed during refreshes. This is particularly useful when dealing with large datasets that are updated frequently.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when building a data model for an e-commerce platform. For instance, you might want to analyze customer feedback data stored in a text column. By applying a 'text contains' filter, you can easily extract rows where customer comments mention specific keywords, such as 'product quality' or 'delivery issues', and then perform further analysis on these filtered results.
