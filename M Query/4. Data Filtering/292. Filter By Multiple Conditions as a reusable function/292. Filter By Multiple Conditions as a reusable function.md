# Filter By Multiple Conditions as a reusable function

**Description:**
This transformation allows for filtering data based on multiple conditions, making it reusable across various data pipelines. It enables users to apply complex filtering logic to their data, enhancing data analysis and processing capabilities.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when analyzing customer data. For instance, a marketing team might want to filter customers based on multiple criteria such as age group (18-35), location (specific cities), and purchase history (customers who have made repeat purchases). This transformation enables the team to easily apply these filters and analyze the targeted customer segment.

### Core M Functions Used
`Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"John", 25, "New York"},
        {"Mary", 31, "Los Angeles"},
        {"David", 42, "New York"},
        {"Jane", 28, "Chicago"}
    }, {"Name", "Age", "City"}),
    FilteredRows = Table.SelectRows(Source, each [Age] > 30 and [City] = "New York")
    in
    FilteredRows
```
