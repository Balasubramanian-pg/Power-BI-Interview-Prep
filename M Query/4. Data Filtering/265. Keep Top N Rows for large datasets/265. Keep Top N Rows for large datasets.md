# Keep Top N Rows for Large Datasets

**Description:**
This transformation allows you to limit the number of rows in a large dataset to the top N rows based on a specific order. It is useful for handling large datasets where only a subset of rows is needed for analysis or processing. By keeping only the top N rows, you can significantly reduce the size of your dataset and improve performance in your data pipeline.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when you need to process a large dataset but only require a certain number of top rows for analysis. For example, if you're analyzing sales data and want to focus on the top 10 products by sales volume, you can use this transformation to keep only those top 10 rows.
