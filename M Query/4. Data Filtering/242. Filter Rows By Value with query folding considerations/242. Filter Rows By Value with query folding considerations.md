# Filter Rows By Value with Query Folding Considerations

**Description:**
This transformation filters rows in a table based on specific values. It is commonly used to subset data based on conditions that must be met for the data to be relevant. The goal is to reduce the dataset to only the rows that match certain criteria, which can improve performance and data quality. This is particularly useful when working with large datasets where not all data is necessary for analysis.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when ingesting data from a database or a file. For example, when loading sales data, you might only be interested in sales from a specific region or above a certain threshold. By filtering the data early in the pipeline, you can reduce the amount of data that needs to be processed, which can lead to significant performance improvements.
