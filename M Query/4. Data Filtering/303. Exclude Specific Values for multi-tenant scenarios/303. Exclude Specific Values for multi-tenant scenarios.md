# Exclude Specific Values for Multi-Tenant Scenarios

**Description:**
This transformation excludes specific values from a dataset, which is particularly useful in multi-tenant scenarios where certain values need to be filtered out to maintain data privacy or segregation.

### Typical Scenario
In a multi-tenant data warehouse, there are often scenarios where certain tenants need to be excluded from analysis due to data privacy or business requirements. For instance, a company may have a data pipeline that aggregates sales data across different regions, but needs to exclude sales data from a specific region or set of regions.

### Core M Functions Used
`Table.SelectRows`

```powerquery
let
    Source = Table.FromRows({
        {"Tenant A", "Value 1"},

        {"Tenant C", "Value 3"}
    }, {"Tenant", "Value"}),
    ExcludedTenants = {"Tenant B"},
    FilteredRows = Table.SelectRows(Source, each not List.Contains(ExcludedTenants, [Tenant]))
    in
    FilteredRows
```
