# Filter By Date Range as a reusable function

**Description:**
This use case demonstrates how Power Query (M) can be used to filter data by a specified date range. It allows users to create a reusable function that can be applied across different data sets. The transformation enables efficient data filtering, making it easier to analyze data within specific time frames.

### Typical Scenario
In a real data pipeline, this transformation would be used when there is a need to analyze sales data for a specific quarter. For instance, a business analyst might want to view sales data for Q2 of 2022 to assess performance. By using a reusable function to filter by a date range, the analyst can easily apply this filter across multiple data sets, such as sales reports and customer feedback, to gain a comprehensive understanding of business performance during that period.
