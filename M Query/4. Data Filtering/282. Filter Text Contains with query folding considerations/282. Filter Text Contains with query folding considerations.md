# Filter Text Contains with Query Folding Considerations

**Description:**

In a real-world data pipeline, this transformation would be used when you need to filter large datasets based on specific text conditions. For example, in an e-commerce data pipeline, you might want to filter product descriptions to only include products whose descriptions contain a certain keyword, such as 'electronic'.
