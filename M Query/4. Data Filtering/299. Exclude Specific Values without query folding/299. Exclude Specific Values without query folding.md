# Exclude Specific Values without Query Folding

**Description:**
This transformation excludes specific values from a data set without relying on query folding, which can be useful when dealing with data sources that do not support query folding. It allows for more flexible data manipulation and filtering.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when you need to filter out specific values from a dataset that is being ingested from a data source that doesn't support query folding, such as certain web APIs or data lakes.
