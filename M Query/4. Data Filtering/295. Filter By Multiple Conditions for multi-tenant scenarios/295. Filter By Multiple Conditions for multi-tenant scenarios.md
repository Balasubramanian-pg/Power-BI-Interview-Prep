# Filter By Multiple Conditions for Multi-Tenant Scenarios

**Description:**
This transformation enables filtering data based on multiple conditions, which is particularly useful in multi-tenant scenarios where data needs to be segregated or analyzed based on various criteria. It allows for dynamic and flexible data processing, ensuring that only relevant data is extracted or analyzed.

### Typical Scenario
In a multi-tenant data environment, such as a SaaS application, this transformation can be used to filter customer data based on multiple conditions like region, subscription plan, and usage level. For instance, a business analyst might want to analyze customer data for clients in North America who are on a premium plan and have exceeded a certain usage threshold.

`Table.SelectRows`

    4. **Filtered Data**: The result is a table that includes only the rows for tenants in North America who are on a premium plan and have a usage level greater than 100.
