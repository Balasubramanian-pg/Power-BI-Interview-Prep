# Filter By Date Range for large datasets

**Description:**
This transformation filters large datasets based on a specified date range. It is commonly used to narrow down data for analysis or reporting purposes. By applying a date range filter, you can efficiently reduce the size of your dataset and focus on relevant information.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when analyzing sales data. For instance, a business might want to analyze sales performance over a specific quarter. They would use the 'Filter By Date Range' transformation to filter sales data to only include records from the desired quarter, making it easier to generate accurate reports and insights.

### Core M Functions Used
`Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"2022-01-01", "John", 100},
        {"2022-06-01", "Jane", 200},
        {"2022-12-01", "Bob", 300}
    }, {"Date", "Name", "Amount"}),
    StartDate = #"2022-01-01",
    EndDate = #"2022-12-31",
    FilteredRows = Table.SelectRows(Source, each StartDate <= [Date] and [Date] <= EndDate)
    in
    FilteredRows
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
    1. **Define the Date Range**: Specify the start and end dates for the filter.
    2. **Apply the Filter**: Use the `Table.SelectRows` function to create a new table that only includes rows where the date falls within the specified range.
    3. **Return the Filtered Table**: The resulting table contains only the rows that meet the date criteria, which can then be used for further analysis or processing.
