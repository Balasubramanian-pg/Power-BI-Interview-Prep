# Remove Null Records Optimized for Incremental Refresh

**Description:**
This transformation removes rows with null values from a table, optimizing it for incremental refresh. It ensures that only valid data is processed, improving efficiency and reducing errors in the data pipeline.

### Typical Scenario
In a real-world data pipeline, this transformation is used when ingesting data from a source that may contain null or empty records. For example, when loading customer data from a database, null records might be present due to incomplete data imports or data cleansing issues. Removing these null records is crucial before loading the data into a data warehouse or performing further transformations.

`Table.SelectRows`


    2. **Selection**: Use the `Table.SelectRows` function to filter out rows where any of the columns contain null values.
