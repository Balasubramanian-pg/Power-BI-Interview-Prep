# Filter Rows By Value for Large Datasets

**Description:**
This transformation filters rows in a large dataset based on specific values. It is commonly used to subset data for further analysis or processing. By applying filter conditions, users can focus on relevant data, improving query performance and data quality.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when dealing with massive datasets where not all data needs to be processed. For instance, in a sales data pipeline, you might want to filter rows to only include sales from a specific region or time period.

`Table.SelectRows`
