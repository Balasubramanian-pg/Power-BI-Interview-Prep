# Filter Text Contains for Large Datasets

**Description:**
This transformation filters large datasets to only include rows where a specified text column contains a particular string or substring. It's useful for narrowing down data based on specific text criteria. The transformation is commonly used in data cleaning and preprocessing steps.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when you need to analyze a subset of data that matches certain text patterns. For example, you might want to filter customer feedback data to only include comments that mention a specific product or feature.

### Core M Functions Used
`Table.SelectRows`, `Text.Contains`

```powerquery
    FilteredRows = Table.SelectRows(Source, each Text.Contains([Feedback], "Product A", [IgnoreCase = true]))
    in
    FilteredRows
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
    1. Start with a table that contains a text column you want to filter on.
    2. Use the Table.SelectRows function to create a new table that includes only the rows where the text column contains the specified string or substring.
    3. The Text.Contains function is used within Table.SelectRows to perform the text search. This function returns true if the text column contains the specified string, and false otherwise.
    4. The [IgnoreCase = true] parameter makes the search case-insensitive, so it will match 'Product A', 'product a', 'PRODUCT A', etc.
