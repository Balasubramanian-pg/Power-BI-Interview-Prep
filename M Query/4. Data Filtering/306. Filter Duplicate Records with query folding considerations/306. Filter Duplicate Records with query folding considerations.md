# Filter Duplicate Records with Query Folding Considerations

**Description:**
This transformation removes duplicate records from a table while considering query folding. Query folding allows Power Query to push data processing operations back to the source, improving performance. By removing duplicates, you ensure data integrity and reduce the amount of data being processed.

### Typical Scenario
In a real data pipeline, this transformation is used when ingesting data from a source that may contain duplicate records, such as when extracting customer information from a database. Removing duplicates ensures that each customer appears only once in the dataset, which is crucial for accurate analysis and reporting.
