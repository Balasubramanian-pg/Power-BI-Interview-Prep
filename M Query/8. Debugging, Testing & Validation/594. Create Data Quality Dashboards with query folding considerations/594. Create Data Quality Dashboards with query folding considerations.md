# Create Data Quality Dashboards with Query Folding Considerations

**Description:**
This use case demonstrates how Power Query (M) can be used to create data quality dashboards while considering query folding. Query folding allows Power Query to push data processing operations back to the source, improving performance. This transformation helps ensure data accuracy and reliability by enabling efficient data quality checks.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when building data quality dashboards for business stakeholders. For instance, a company might want to monitor data quality metrics such as data completeness, accuracy, and consistency across different data sources. By using Power Query with query folding considerations, the company can efficiently process large datasets and provide up-to-date data quality insights to stakeholders.

By using query folding, Power Query pushes these operations back to the source, minimizing data processing and improving performance.
