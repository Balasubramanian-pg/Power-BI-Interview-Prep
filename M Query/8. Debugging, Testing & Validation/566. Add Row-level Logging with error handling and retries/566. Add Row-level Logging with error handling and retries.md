# Add Row-level Logging with Error Handling and Retries

**Description:**
This transformation adds row-level logging to a data pipeline, enabling the tracking of errors and retries at the row level. It enhances data quality and pipeline reliability by providing detailed logs of operations performed on each row. This is particularly useful for auditing and troubleshooting purposes.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when integrating data from multiple sources, where data quality varies. For instance, when loading customer data from various systems, the pipeline can log which rows failed to load due to errors, how many retries were attempted, and the final status of each row.

### Core M Functions Used
`Table.SelectRows`, `Table.AddColumn`, `try`, `catch`, `Number.Retries`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"John", 25},
        {"Jane", 30},
        {"Bob", 35}
    }, {"Name", "Age"}),
    AddLogColumn = Table.AddColumn(Source, "Log", each try
        [Name] & " loaded successfully"
    catch
        [Exception.Message] & " with error"),
    RetryCount = Table.AddColumn(AddLogColumn, "Retry Count", each try
        0
    catch
        Number.Retries([Exception]))
    in
    RetryCount
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
    1. **Data Source**: The pipeline starts with a table of data.
    2. **Add Logging Column**: A new column is added to the table where each row's value is determined by a try-catch block. If the operation on the row is successful, it logs a success message; otherwise, it logs the error message.
    3. **Retry Mechanism**: Another column is added to track the number of retries for each row. This can be achieved by using the `Number.Retries` function within the try-catch block.
    4. **Result**: The final output is a table with the original data plus two additional columns for logging and retry counts, providing a detailed audit trail of the operations performed on each row.
