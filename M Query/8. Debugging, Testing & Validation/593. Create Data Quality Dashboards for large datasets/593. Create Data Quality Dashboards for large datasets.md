# Create Data Quality Dashboards for large datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to create data quality dashboards for large datasets, enabling data analysts to monitor and improve data quality.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when data quality is a top priority, such as in financial data processing or healthcare data analysis, where data accuracy and reliability are crucial.

### Core M Functions Used
`Table.SelectRows`, `Table.TransformColumns`, `Table.Group`

### Code Source Execution
```powerquery
let
    data = Table.FromRows({
        {"John", 25, "USA"},
        {"Mary", 31, "Canada"},
        {"David", 42, "UK"}
    }, {"Name", "Age", "Country"}),
    qualityCheck = Table.TransformColumns(data, {{"Age", each if _ > 0 then _ else null}}),
    summary = Table.Group(qualityCheck, {"Country"}, {{"Count", each Table.RowCount(_), type nullable number}})
    in
    summary
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
    1. Load the data into a table.
    2. Perform a quality check on the 'Age' column to ensure values are greater than 0.
    3. Group the data by 'Country' and count the number of rows for each group.
    The resulting table provides a summary of data quality by country.
