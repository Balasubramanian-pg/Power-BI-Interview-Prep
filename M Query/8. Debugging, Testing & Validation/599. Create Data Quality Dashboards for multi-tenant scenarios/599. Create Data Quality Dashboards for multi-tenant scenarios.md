# Create Data Quality Dashboards for multi-tenant scenarios

**Description:**
This use case demonstrates how Power Query (M) can be used to create data quality dashboards for multi-tenant scenarios, enabling the assessment and monitoring of data quality across multiple tenants.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when there are multiple tenants with different data sources and structures, and the need arises to evaluate and visualize data quality metrics such as data completeness, accuracy, and consistency across all tenants.

### Core M Functions Used
`Table.SelectRows`, `Table.TransformColumns`, `Table.Group`

### Code Source Execution
```powerquery
let
    // Sample data
    data = Table.FromList({
        {"Tenant1", "John", 25},
        {"Tenant1", "Alice", 30},
        {"Tenant2", "Bob", 35},
        {"Tenant2", "Eve", 40}
    }, Splitter.SplitByNothing(), null, null, ExtraValues.Error),
    // Add column for data quality evaluation
    dataWithQuality = Table.AddColumn(data, "Quality", each if [Age] > 30 then "Good" else "Bad"),
    // Group by tenant and evaluate data quality
    dataQualityByTenant = Table.Group(dataWithQuality, {"Tenant"}, {
        {"Count", each Table.RowCount(_), type nullable number},
        {"Good Quality Count", each Table.RowCount(Table.SelectRows(_, "Quality", "Good")), type nullable number}
    })
    in
    dataQualityByTenant
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
    1. **Data Ingestion**: Start by ingesting data from various tenants into a single table.
    2. **Data Quality Evaluation**: Add a new column to the table to evaluate the quality of each data row based on predefined criteria (e.g., age > 30).
    3. **Grouping by Tenant**: Group the data by tenant.
    4. **Aggregation**: For each tenant, count the total number of rows and the number of rows with good quality data.
    5. **Visualization**: Finally, visualize these metrics in a dashboard to monitor data quality across tenants.
