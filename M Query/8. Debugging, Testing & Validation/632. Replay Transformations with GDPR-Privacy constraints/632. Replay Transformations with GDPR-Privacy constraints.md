# Replay Transformations with GDPR/Privacy constraints

**Description:**
This use case demonstrates how Power Query (M) can be used for replay transformations while adhering to GDPR/privacy constraints. It ensures that sensitive data is handled appropriately during transformations.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial when replaying data for analytics or debugging purposes while ensuring that personally identifiable information (PII) is protected. For instance, a company might need to replay customer interactions for troubleshooting, but must do so without exposing sensitive customer data.
