# Detect Null Injection for Large Datasets

**Description:**
This transformation detects and handles null injection attacks in large datasets. It identifies rows with null or missing values and provides options for handling them. This is particularly useful in data pipelines where data quality and integrity are crucial.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when ingesting large datasets from untrusted sources, such as customer uploads or third-party data providers. For instance, a company may receive a large dataset from a vendor, and before processing it further, they want to detect any potential null injection attacks that could compromise their analysis or reporting.

`Table.SelectRows`, `Table.ColumnNames`, `List.Contains`
