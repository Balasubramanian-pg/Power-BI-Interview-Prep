# Adding Row-level Logging for Large Datasets in Power Query

**Description:**
This transformation enables the addition of row-level logging to large datasets in Power Query, allowing for detailed tracking and auditing of data changes. It is particularly useful for maintaining data integrity and compliance with regulatory requirements. By incorporating row-level logging, users can monitor data modifications and ensure transparency across the data pipeline.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when loading data from a source system to a data warehouse or data mart. For instance, when loading customer data from a CRM system to a data warehouse, row-level logging can be used to track changes to customer records, such as updates to contact information or changes to account status. This provides a clear audit trail for data changes, enabling data stewards to monitor data quality and ensure compliance with data governance policies.
