# Import XML Data for Large Datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to efficiently import XML data for large datasets. It enables users to extract, transform, and load XML data into a usable format for analysis.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial when dealing with large volumes of data stored in XML format, such as data from web services, logs, or legacy systems. For instance, a company might need to import XML data from a web service that provides daily sales reports. The XML data would be imported, transformed, and then loaded into a database or data warehouse for further analysis and reporting.

### Core M Functions Used
`Xml.Document`, `Table.FromList`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Xml.Document(File.Contents("C:\Path\To\Your\File.xml")),
    Root = Source{[Root]},
    Items = Root[Items]{0}[Item],
    TableFromList = Table.FromList(Items, Splitter.SplitByNothing(), null, null, ExtraValues.Error),
    ExpandedTable = Table.ExpandRecordColumn(TableFromList, "Column1", {"Attribute1", "Attribute2"}, {"Attribute1", "Attribute2"})
    in
    ExpandedTable
```

The transformation works as follows:
1. **Read XML File**: The `Xml.Document` function reads the XML file.
2. **Navigate XML Structure**: Navigate through the XML structure to reach the desired data level.
3. **Convert to Table**: Use `Table.FromList` to convert the XML data into a table format.
4. **Expand Records**: Use `Table.ExpandRecordColumn` to expand the record columns into separate columns.
5. **Select and Filter Data**: Optionally, use `Table.SelectRows` to filter the data based on specific conditions.
