# Load JSON File optimized for incremental refresh

**Description:**
This transformation optimizes the loading of JSON files for incremental refresh, enabling efficient updates to data without reloading the entire file.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial when dealing with large JSON files that are frequently updated, such as log files or social media data feeds. Incremental refresh allows for near-real-time data analysis without the overhead of completely reloading the data.

### Core M Functions Used
`Json.Document`, `Table.SelectRows`, `Table.SelectColumns`

```powerquery
    SelectedRows
```
