# Connect to REST API as a reusable function

**Description:**
This use case demonstrates how Power Query (M) can be used to connect to a REST API as a reusable function, allowing for easy data retrieval and manipulation.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when connecting to external data sources via REST APIs, such as retrieving sales data from an e-commerce platform or fetching weather data from a public API.

### Core M Functions Used
`Web.Contents`

### Code Source Execution
```powerquery
let
    apiUrl = "https://api.example.com/data",
    apiKey = "your_api_key",
    headers = [#"Authorization" = "Bearer " & apiKey],
    data = Web.Contents(apiUrl, [Headers=headers]),
    jsonData = Json.Document(data)
in
    jsonData
```

### Example Execution Logic Step-By-Step
The transformation works by first defining the API URL and any required authentication headers. It then uses the Web.Contents function to send a GET request to the API, passing in the headers. The response data is then parsed as JSON using the Json.Document function, allowing for easy navigation and manipulation of the data in Power Query.
