# Connect to PostgreSQL with Query Folding Considerations

**Description:**
This use case demonstrates how Power Query (M) can be used to connect to a PostgreSQL database while considering query folding. Query folding allows Power Query to push data processing operations down to the source database, improving performance. This is particularly useful when working with large datasets.

### Typical Scenario
In a real data pipeline, this transformation would be used when connecting to a PostgreSQL database to retrieve specific data. For example, a business analyst might need to fetch sales data for a specific region from a PostgreSQL database. By using query folding, the data processing operations can be pushed down to the database, reducing the amount of data that needs to be transferred and processed in Power Query.
