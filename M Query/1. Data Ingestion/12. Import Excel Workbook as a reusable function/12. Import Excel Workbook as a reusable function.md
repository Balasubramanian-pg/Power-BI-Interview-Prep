# Import Excel Workbook as a reusable function

**Description:**
This use case demonstrates how Power Query (M) can be used to import an Excel Workbook as a reusable function, allowing for efficient data extraction and transformation.

### Typical Scenario
In a real data pipeline, this transformation would be used when there is a need to regularly import data from an Excel Workbook, such as when dealing with weekly or monthly reports that are submitted in Excel format.

### Core M Functions Used
`Excel.Workbook`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Excel.Workbook(File.Contents("C:\Path\To\Workbook.xlsx"), null, true),
    Sheet1 = Source{[0]},
    Data = Sheet1{Data},
    RenameColumns = Table.RenameColumns(Data, {{ "Column1", "NewColumnName" }})
    in
    RenameColumns
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
1. The Excel.Workbook function is used to import the Excel Workbook. 
2. The specific sheet to be imported is selected. 
3. The data is extracted from the sheet. 
4. The columns are renamed as needed using Table.RenameColumns.
