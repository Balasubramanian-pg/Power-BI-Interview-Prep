# Connect to MySQL optimized for incremental refresh

**Description:**
This use case demonstrates how Power Query (M) can be used to connect to a MySQL database and optimize the data retrieval process for incremental refresh. This approach enables efficient data updates by only retrieving new or changed data.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when there is a need to regularly update data from a MySQL database, such as daily sales reports or customer information. For instance, an e-commerce company might use this to fetch daily sales data from their MySQL database, processing only new sales records since the last update.

`MySql.Database`, `Table.SelectRows`

```powerquery
    IncrementalRefresh
```
