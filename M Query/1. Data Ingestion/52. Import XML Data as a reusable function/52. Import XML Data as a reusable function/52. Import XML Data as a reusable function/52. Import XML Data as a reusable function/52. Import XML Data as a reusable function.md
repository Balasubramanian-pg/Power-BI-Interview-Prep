# Import XML Data as a Reusable Function

**Description:**
This use case demonstrates how Power Query (M) can be used to import XML data as a reusable function, allowing for efficient data transformation and reuse across different queries.

In a real-world data pipeline, this transformation would be used when there is a need to regularly import data from XML sources, such as retrieving data from legacy systems, web services, or file systems, and then transforming it into a usable format for analysis or reporting.

`Xml.Document`, `Table.FromList`, `Table.SelectRows`

    data = xml[Root]\"Data\"[Table]\"Table1\".{0}[#\"Element1\", #\"Element2\"] in
