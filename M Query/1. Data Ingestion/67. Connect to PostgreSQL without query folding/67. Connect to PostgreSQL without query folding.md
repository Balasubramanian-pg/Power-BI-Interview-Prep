# Connect to PostgreSQL without query folding

**Description:**
This use case demonstrates how Power Query (M) can be used to connect to a PostgreSQL database without query folding. It allows for the retrieval of data directly from PostgreSQL while preserving the flexibility of Power Query transformations.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when you need to fetch data from a PostgreSQL database and apply complex filtering or transformations that cannot be folded back to the database. For instance, when you need to perform row-level calculations or string manipulations that are not supported by the database.
