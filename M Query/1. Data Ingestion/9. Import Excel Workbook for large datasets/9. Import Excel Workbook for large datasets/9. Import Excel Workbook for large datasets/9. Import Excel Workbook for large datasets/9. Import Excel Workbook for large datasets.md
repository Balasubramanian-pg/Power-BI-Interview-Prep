# Import Excel Workbook for large datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to import Excel workbooks for handling large datasets. It enables efficient data extraction and transformation from Excel files.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when dealing with large datasets stored in Excel workbooks. For instance, a business analyst might need to import sales data from a monthly Excel report into a data warehouse for further analysis.

### Core M Functions Used
`Excel.Workbook`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Excel.Workbook(File.Contents("C:\Path\To\Workbook.xlsx"), null, true),
    Data = Source{[0]}{[Data]},
    SelectedData = Table.SelectRows(Data, each [ColumnName] <> null)
in
    SelectedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
1. **Read Excel Workbook**: The `Excel.Workbook` function reads the contents of an Excel file.
2. **Select Data**: The `[0]{[Data]}` syntax selects the first sheet's data.
3. **Filter Rows**: The `Table.SelectRows` function filters rows based on conditions, e.g., excluding rows with null values in a specific column.
