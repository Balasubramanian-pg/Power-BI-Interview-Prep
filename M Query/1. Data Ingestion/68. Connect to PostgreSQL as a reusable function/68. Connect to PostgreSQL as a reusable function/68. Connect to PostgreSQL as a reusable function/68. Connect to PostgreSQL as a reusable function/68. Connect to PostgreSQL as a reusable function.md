# Connect to PostgreSQL as a reusable function

**Description:**
This use case demonstrates how Power Query (M) can be used to connect to a PostgreSQL database as a reusable function, allowing for easy data retrieval and manipulation.

### Typical Scenario
In a real data pipeline, this transformation would be used when there is a need to frequently connect to a PostgreSQL database to retrieve or update data, such as in a daily data refresh or when integrating data from multiple sources.

### Core M Functions Used
`Table.SelectRows`

### Code Source Execution
```powerquery
let
    PostgreSQLConn = GetPostgreSQLConnection(),
    Query = "SELECT * FROM your_table",
    Data = Sql.Database(PostgreSQLConn, "your_database", [Query=Query])
in Data
```

### Example Execution Logic Step-By-Step
The transformation works by first establishing a connection to the PostgreSQL database using the GetPostgreSQLConnection function. Then, a SQL query is defined to select data from a specific table. The Sql.Database function is used to execute the query and retrieve the data, which is then returned as a table.
