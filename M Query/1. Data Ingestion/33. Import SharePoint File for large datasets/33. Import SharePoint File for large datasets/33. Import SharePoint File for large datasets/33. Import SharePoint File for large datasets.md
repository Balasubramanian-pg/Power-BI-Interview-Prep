# Import SharePoint File for large datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to import SharePoint files for large datasets, enabling efficient data retrieval and processing.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when dealing with large datasets stored in SharePoint, such as retrieving data from SharePoint lists or libraries for data analysis and reporting.

### Core M Functions Used
`Table.SelectRows`, `SharePoint.Files`, `Folder.Contents`

### Code Source Execution
```powerquery
let
    Source = SharePoint.Files("https://your-sharepoint-site.sharepoint.com/your-library"),
    FolderContents = Table.SelectRows(Source, each [Kind] = "Folder"),
    FileContents = Table.SelectRows(Source, each [Kind] = "File")
in FolderContents
```

### Example Execution Logic Step-By-Step
The transformation works by first connecting to the SharePoint library using the SharePoint.Files function. It then uses Table.SelectRows to filter the contents into folders and files. The Folder.Contents function can then be used to recursively retrieve files from the folders.
