# Import XML Data Optimized for Incremental Refresh

**Description:**
This use case demonstrates how Power Query (M) can be used to import XML data in a way that optimizes incremental refresh. It allows for efficient updates to data by only refreshing the changed parts. This approach is particularly useful for large datasets where a full refresh is unnecessary.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial when dealing with large XML files that are frequently updated. For instance, a financial institution might receive daily XML updates of transactions. By using incremental refresh, the institution can quickly update its database without reprocessing the entire file, saving time and resources.

### Core M Functions Used
`Xml.Document`, `Table.SelectRows`, `Table.SelectColumns`

    in
    FilteredRows
