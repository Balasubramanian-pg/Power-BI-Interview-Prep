# Import SharePoint File without query folding

**Description:**
This use case demonstrates how Power Query (M) can be used to import a SharePoint file without query folding, allowing for flexible data retrieval and processing.

### Typical Scenario
In a real data pipeline, this transformation would be used when you need to fetch files from SharePoint for data processing or analysis, and the files are not too large to be handled by Power Query directly.

### Core M Functions Used
`Web.Contents`, `Table.FromList`

### Code Source Execution
```powerquery
let
    url = "https://your-sharepoint-site.sharepoint.com/your-library/your-file.csv",
    fileContents = Web.Contents(url),
    csvData = Table.FromList(Splitter.SplitByNothing() {fileContents}, null, null, ExtraValues.Error),
    data = Table.TransformColumns(csvData, {"Column1", each _}) in data
```

### Example Execution Logic Step-By-Step
The transformation works by first specifying the URL of the SharePoint file you want to import. It then uses the Web.Contents function to fetch the file contents. The file contents are then converted into a table using Table.FromList. Finally, the table is transformed to ensure data is properly formatted for use in your data pipeline.
