# Read CSV File as a Reusable Function

**Description:**
This use case demonstrates how Power Query (M) can be used to create a reusable function for reading CSV files. In a real-world data pipeline, this transformation would be used when there is a need to frequently import data from CSV files. For example, a company might receive daily sales data in CSV format from an external source. The data team can use this reusable function to automate the process of reading these CSV files, transforming the data as needed, and loading it into their data warehouse for analysis.

### Core M Functions Used
`File.Contents`, `Csv.Document`

### Code Source Execution
```powerquery
let
    ReadCSV = (filePath as text) =>
        let
            fileContents = File.Contents(filePath),
            csvDocument = Csv.Document(fileContents, [Delimiter=",", Encoding=1252])
        in
            csvDocument
in
    ReadCSV
```
