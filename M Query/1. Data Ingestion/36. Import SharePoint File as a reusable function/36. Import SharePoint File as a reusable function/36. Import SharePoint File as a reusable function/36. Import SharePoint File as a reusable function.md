# Import SharePoint File as a reusable function

**Description:**
This use case demonstrates how Power Query (M) can be used to import a SharePoint file as a reusable function, allowing for efficient data retrieval and transformation.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when there is a need to regularly import data from a SharePoint file, such as a report or a dataset, into a data model for analysis and visualization.

### Core M Functions Used
`Web.Contents`, `SharePoint.Files`

### Code Source Execution
```powerquery
let
    Source = SharePoint.Files("https://your-sharepoint-site.sharepoint.com"),
    FileContents = Source{[Name="YourFileName"]}[Content]
    in
    FileContents
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
1. The SharePoint.Files function is used to connect to the SharePoint site and retrieve a list of files.
2. The Web.Contents function is used to retrieve the contents of the specified file.
3. The file contents are then returned as a binary value, which can be transformed and loaded into a data model.
