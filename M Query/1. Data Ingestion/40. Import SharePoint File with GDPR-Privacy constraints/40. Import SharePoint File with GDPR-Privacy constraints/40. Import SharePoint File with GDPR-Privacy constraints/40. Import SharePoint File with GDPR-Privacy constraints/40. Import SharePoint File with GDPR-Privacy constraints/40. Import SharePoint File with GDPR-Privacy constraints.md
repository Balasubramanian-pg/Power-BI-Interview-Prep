# Import SharePoint File with GDPR/Privacy constraints

**Description:**
This use case demonstrates how Power Query (M) can be used for importing SharePoint files while adhering to GDPR/privacy constraints. It ensures that sensitive data is handled appropriately during the import process.

In a real data pipeline, this transformation would be used when an organization needs to import files from SharePoint for data analysis while ensuring compliance with GDPR/privacy regulations. For instance, when importing employee data, the transformation can be used to anonymize or filter out sensitive information.
