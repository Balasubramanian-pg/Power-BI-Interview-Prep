# Connect to REST API for large datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to connect to a REST API for retrieving large datasets. It enables data extraction from REST APIs, which can be crucial for integrating external data sources into Power BI or other data analysis tools. In a real-world data pipeline, this transformation would be used when a business needs to integrate data from an external service that only provides access through a REST API. For example, a company might use this to fetch large datasets from a third-party service for analysis in Power BI.

`Web.Contents`, `Json.Document`, `Table.FromList`

### Code Source Execution
```powerquery
let
    apiUrl = "https://api.example.com/large-dataset",
    apiKey = "your_api_key_here",
    headers = [#"Authorization" = "Bearer " & apiKey],
    data = Web.Contents(apiUrl, [Headers=headers]),
    jsonData = Json.Document(data),
    dataset = Table.FromList(jsonData, Splitter.SplitByNothing(), {"data"}) in
    dataset
```
