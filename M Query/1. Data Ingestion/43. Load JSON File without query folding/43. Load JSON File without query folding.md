# Load JSON File without Query Folding

**Description:**
This use case demonstrates how Power Query (M) can be used to load a JSON file without query folding. It is useful when the JSON file needs to be processed in its entirety, without any optimization or folding of queries.

### Typical Scenario
In a real data pipeline, this transformation would be used when ingesting data from a JSON file that is not optimized for query folding, such as when dealing with complex or nested JSON structures.

### Core M Functions Used
`Json.Document`, `Table.FromList`

### Code Source Execution
```powerquery
let
    Source = Json.Document(File.Contents("C:\\path\\to\\file.json")),
    ConvertedToTable = Table.FromList(Source, Splitter.SplitByNothing(), null, null, ExtraValues.Error),
    ExpandedTable = Table.ExpandRecordColumn(ConvertedToTable, "Column1", {"column1", "column2"}, {"column1", "column2"})
in
    ExpandedTable
```

### Example Execution Logic Step-By-Step
The transformation works by first loading the JSON file using the Json.Document function. The resulting JSON object is then converted to a table using Table.FromList. Finally, the table is expanded to include the individual columns of the JSON object using Table.ExpandRecordColumn.
