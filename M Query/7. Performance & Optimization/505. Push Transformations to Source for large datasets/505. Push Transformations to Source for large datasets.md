# Push Transformations to Source for large datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to push data transformations back to the source for large datasets, improving performance by reducing the amount of data that needs to be processed in the Power Query environment.

### Typical Scenario
In a real-world data pipeline, this transformation is particularly useful when dealing with massive datasets that are too large to handle efficiently within the Power Query environment. For instance, when working with big data from IoT devices or large-scale financial transactions, pushing transformations to the source (e.g., a database) can significantly reduce the data volume that needs to be transferred and processed, thereby speeding up the overall data processing and analysis.
