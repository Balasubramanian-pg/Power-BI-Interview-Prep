# Parallelize File Processing with GDPR/Privacy constraints

**Description:**
This use case demonstrates how Power Query (M) can be used for parallelizing file processing while adhering to GDPR/privacy constraints. It enables efficient handling of large datasets from multiple files, ensuring that sensitive information is processed in compliance with data protection regulations.

### Typical Scenario
In a real data pipeline, this transformation would be used when processing large volumes of sensitive data from multiple files. For instance, a company needs to process customer data from various sources, ensuring that GDPR/privacy constraints are met. By parallelizing file processing, the company can significantly reduce processing time while maintaining data integrity and compliance.

### Core M Functions Used
`Table.SelectRows`, `Table.Combine`, `List.Transform`

### Code Source Execution
```powerquery
let
    // Source files
    files = {"file1.csv", "file2.csv", "file3.csv"},
    // Parallel processing function
    processFile = (file as text) =>
        let
            // Read file
            data = Table.FromRows(File.Contents(file)),
            // Apply GDPR/privacy transformations
            transformedData = Table.SelectRows(data, each GDPRCheck())
        in
            transformedData,
    // Parallelize file processing
    results = List.Transform(files, processFile),
    // Combine results
    combinedData = Table.Combine(results)
    in
        combinedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
1. **Identify Source Files**: List the files to be processed.
2. **Define Processing Function**: Create a function that reads a file, applies necessary GDPR/privacy transformations using `Table.SelectRows`, and returns the transformed data.
3. **Parallelize Processing**: Use `List.Transform` to apply the processing function to each file in parallel.
4. **Combine Results**: Use `Table.Combine` to merge the results from each file into a single table.
This approach ensures efficient and compliant processing of sensitive data.
