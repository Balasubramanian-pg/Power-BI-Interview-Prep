# Enable Query Folding as a reusable function

**Description:**
This use case demonstrates how Power Query (M) can be used to enable query folding as a reusable function, allowing for more efficient data processing by pushing operations back to the source.

### Typical Scenario
In a real data pipeline, enabling query folding as a reusable function would be used when you need to frequently apply a set of operations to large datasets, such as filtering and sorting, and you want to minimize the amount of data transferred between the source and the Power Query environment.
