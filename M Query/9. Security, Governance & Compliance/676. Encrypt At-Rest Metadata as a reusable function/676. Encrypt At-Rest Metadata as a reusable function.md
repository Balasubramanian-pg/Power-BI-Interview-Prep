# Encrypt At-Rest Metadata as a reusable function

**Description:**
This use case demonstrates how Power Query (M) can be used to encrypt metadata at rest as a reusable function, enhancing data security and privacy.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when sensitive metadata, such as data source credentials or personal identifiable information, needs to be stored securely. For instance, a data engineer might use this function to encrypt metadata before storing it in a file or database, ensuring that even if the storage medium is compromised, the metadata remains protected.
