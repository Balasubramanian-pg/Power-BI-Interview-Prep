# Encrypt At-Rest Metadata for large datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to encrypt metadata for large datasets at rest, ensuring data security and compliance with regulatory requirements. By encrypting metadata, organizations can protect sensitive information such as data source credentials, data lineage, and data catalog information. This transformation is crucial for securing data warehouses, data lakes, and other data storage systems.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when a company needs to store large datasets securely, such as in a cloud-based data warehouse or data lake. For instance, a financial institution may need to encrypt metadata for customer data stored in a data warehouse to comply with financial regulations and protect customer information.

`Table.SelectRows`, `Binary.ToText`, `Text.FromBinary`
