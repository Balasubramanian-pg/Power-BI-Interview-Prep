# Group Rows By Category without query folding

**Description:**
This transformation groups rows by a specific category without relying on query folding, which can be useful when dealing with complex data sources or custom aggregation needs.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when you need to aggregate data by categories that aren't directly supported by the data source's query folding capabilities, such as when performing complex aggregations or when the data source doesn't support grouping.
