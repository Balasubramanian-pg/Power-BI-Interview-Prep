# Group Rows By Category for Multi-Tenant Scenarios

**Description:**
This transformation groups rows by a specific category, which is particularly useful in multi-tenant scenarios where data needs to be aggregated and analyzed by tenant or category. It enables efficient data organization and analysis across multiple tenants.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when dealing with multi-tenant data sources, such as a single database table containing sales data from different regions or companies. For instance, an e-commerce platform with multiple vendors might need to group sales data by vendor to analyze performance.
