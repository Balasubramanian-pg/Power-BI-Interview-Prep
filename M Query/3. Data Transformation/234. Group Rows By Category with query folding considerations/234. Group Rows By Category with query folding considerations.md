# Group Rows By Category with Query Folding Considerations

**Description:**
This transformation groups rows by a specific category, allowing for efficient data aggregation and analysis. It is particularly useful when dealing with large datasets and optimizing query performance. By leveraging query folding, Power Query can push the grouping operation to the source, reducing the amount of data that needs to be processed.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when analyzing sales data by region. For instance, a company may want to group sales rows by region, calculate the total sales for each region, and then analyze the results to identify top-performing regions.
