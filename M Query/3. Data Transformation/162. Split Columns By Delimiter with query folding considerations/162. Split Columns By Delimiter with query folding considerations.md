# Split Columns By Delimiter with Query Folding Considerations

**Description:**
This transformation splits a column into multiple columns based on a specified delimiter. It is useful for data cleansing and preparation, particularly when dealing with data that contains comma-separated values or other delimited data. By considering query folding, this transformation can be optimized for better performance.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when ingesting data from a CSV file or other delimited text file. For example, a company may receive a monthly sales report in a CSV file where each row represents a sale and each column represents a field such as date, product, and quantity. However, the product column contains multiple values separated by commas, and the company wants to analyze each product separately. In this case, the Split Columns By Delimiter transformation can be used to split the product column into separate columns for each product.
