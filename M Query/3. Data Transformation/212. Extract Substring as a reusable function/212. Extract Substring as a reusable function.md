# Extract Substring as a Reusable Function

**Description:**
This transformation extracts a specific substring from a given text column, allowing for flexible data manipulation and cleaning. It enables users to define the start and end points of the substring, making it a powerful tool for data processing. By creating a reusable function, users can easily apply this transformation across multiple columns or tables.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when dealing with messy data that contains concatenated information. For instance, a column might contain full addresses with city, state, and zip code concatenated together. By extracting the zip code as a substring, users can isolate and process this information for further analysis or reporting.

### Core M Functions Used
`Text.Between`, `Text.Start`, `Text.End`

### Code Source Execution
```powerquery
let
    ExtractSubstring = (text as text, start as text, end as text) =>
        Text.Between(text, start, end),
    Source = #"Sample Table",
    Transformed = Table.TransformColumns(Source, {{"Column1", each ExtractSubstring(_, "start", "end")}})
    in
        Transformed
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
    1. Define a reusable function `ExtractSubstring` that takes three parameters: the input text, the start substring, and the end substring.
    2. Use the `Text.Between` function to extract the substring from the input text, bounded by the start and end substrings.
    3. Apply this function to a specific column in a table using `Table.TransformColumns`.
    4. The result is a new table with the transformed column containing the extracted substrings.
