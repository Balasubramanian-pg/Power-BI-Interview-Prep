# Group Rows By Category optimized for incremental refresh

**Description:**
This transformation optimizes the grouping of rows by category for incremental data refreshes, enhancing performance and efficiency in data processing pipelines.

### Typical Scenario
In a sales data pipeline, when new sales data is added daily, this transformation helps in efficiently grouping and analyzing sales by product categories without reprocessing the entire dataset.

### Example Execution Logic Step-By-Step
The transformation works by first loading the data into Power Query. It then groups the rows by the 'Category' column. For each group, it calculates the total sales using the List.Sum function. This approach allows for efficient grouping and aggregation of data, which is particularly useful when dealing with large datasets and incremental refreshes.
