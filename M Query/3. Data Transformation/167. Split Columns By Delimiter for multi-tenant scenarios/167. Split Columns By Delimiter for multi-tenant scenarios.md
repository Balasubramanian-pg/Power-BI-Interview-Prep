# Split Columns By Delimiter for Multi-Tenant Scenarios

**Description:**
This transformation splits a column into multiple columns based on a specified delimiter, which is particularly useful in multi-tenant scenarios where data from different tenants needs to be processed and analyzed separately.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when ingesting data from a multi-tenant application where certain columns contain tenant-specific data separated by a delimiter. For instance, a column might contain URLs or paths separated by a slash, and splitting this column can help in routing data to the correct tenant for further analysis.
