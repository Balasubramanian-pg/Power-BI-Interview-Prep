# Create Calculated Column as a Reusable Function

**Description:**
This transformation allows users to create a calculated column that can be reused across different queries and data sources. It enables efficient data transformation and reduces redundancy in Power Query (M) scripts. By encapsulating the calculation logic into a function, users can easily apply it to various tables and data sets.

### Typical Scenario
In a real-world data pipeline, this transformation is useful when there are multiple tables that require the same calculation, such as computing the total sales amount or converting date formats. For instance, a company may have different sales data sources (e.g., online, offline, and wholesale) that all need to calculate the total revenue. By creating a reusable function for this calculation, data engineers can avoid duplicating code and make maintenance easier.
