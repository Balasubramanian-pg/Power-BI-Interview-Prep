# Split Columns By Delimiter for large datasets

**Description:**
This transformation splits a column into multiple columns based on a specified delimiter, which is particularly useful for handling large datasets with complex data structures.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when dealing with data imports from legacy systems or external sources where data is not properly normalized. For instance, a column in a dataset might contain comma-separated values that need to be split into separate columns for analysis.

### Core M Functions Used
`Table.SplitColumn`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"John,Smith,25"},
        {"Jane,Doe,30"}
    }, {"Name"}),
    SplitName = Table.SplitColumn(Source, "Name", Splitter.SplitByDelimiter(",", "Each"), {"First Name", "Last Name", "Age"})
    in
    SplitName
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
1. The data is first loaded into a table.
2. The Table.SplitColumn function is used to split the 'Name' column by the comma delimiter.
3. The 'Each' option specifies that the delimiter is applied to each row.
4. The result is a new table with the original 'Name' column split into 'First Name', 'Last Name', and 'Age' columns.
