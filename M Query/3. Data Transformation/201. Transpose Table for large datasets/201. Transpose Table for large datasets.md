# Transpose Table for Large Datasets

**Description:**
This transformation is used to transpose a table, converting rows into columns or vice versa, which is particularly useful when dealing with large datasets that need to be reshaped for analysis or reporting.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when ingesting data from a source system where data is normalized in a way that doesn't match the desired structure for analysis. For example, when dealing with survey data where each question's response is stored in a separate row, transposing the table allows for easier analysis and reporting by having questions as columns and responses as values.

`Table.Transpose`, `Table.SelectRows`

### Example Execution Logic Step-By-Step
The transformation works by first loading the data into a table. Then, it uses the Table.Transpose function to swap the rows and columns. For instance, given a table with two columns (Column1 and Column2) where Column1 contains identifiers (A, B, C) and Column2 contains values (1, 2, 3) for the first row and (4, 5, 6) for the second row, transposing it results in a new table where the original rows become columns and vice versa. The Table.SelectRows function can be used in conjunction to filter or select specific rows or columns during the transformation process.
