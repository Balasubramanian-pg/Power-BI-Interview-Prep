# Pivot Table for Large Datasets using Power Query (M)

**Description:**
This transformation enables efficient data summarization and analysis by pivoting large datasets. It helps in transforming data from a long format to a wide format, making it easier to analyze and visualize. This is particularly useful when dealing with extensive data that needs to be aggregated and summarized by specific columns.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when analyzing sales data. For instance, you might have a large dataset containing sales transactions with columns for date, product, region, and sales amount. By pivoting this data, you can create a summary table that shows total sales by product and region over time, making it easier to identify trends and patterns.
