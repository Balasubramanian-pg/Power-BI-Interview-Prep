# Merging Columns as a Reusable Function in Power Query

**Description:**
This transformation allows users to combine multiple columns into a single column, often for data cleansing or restructuring purposes. It is commonly used to concatenate data from separate fields into a unified format. By creating a reusable function, users can easily apply this transformation across different parts of their data pipeline.

### Typical Scenario
In a real-world data pipeline, merging columns is often necessary when dealing with customer data. For instance, you might have separate columns for first name and last name, and you want to create a full name column. This can be particularly useful for reporting or data visualization purposes where a unified identifier is required.

### Core M Functions Used
`Table.CombineColumns`, `Text.Combine`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"John", "Doe", 30},
        {"Jane", "Smith", 25}
    }, {"FirstName", "LastName", "Age"}),
    MergeColumns = (tbl as table, columnsToMerge as list, newColumnName as text) =>
        Table.CombineColumns(tbl, columnsToMerge, Text.Combine, newColumnName),
    MergedTable = MergeColumns(Source, {"FirstName", "LastName"}, "FullName")
    in
        MergedTable
```
