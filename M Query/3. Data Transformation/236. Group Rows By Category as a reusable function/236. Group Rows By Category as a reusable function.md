# Group Rows By Category as a Reusable Function

**Description:**
This transformation groups rows by a specified category, allowing for aggregated calculations and data summarization. It is useful for organizing and analyzing large datasets. By grouping rows, you can perform various operations such as sum, average, and count on specific columns.

### Typical Scenario
In an e-commerce data pipeline, you might want to group sales data by product category to analyze total sales, average order value, and top-performing products within each category.
