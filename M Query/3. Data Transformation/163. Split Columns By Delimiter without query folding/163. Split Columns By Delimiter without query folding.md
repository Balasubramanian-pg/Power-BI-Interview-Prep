# Split Columns By Delimiter without query folding

**Description:**
This use case demonstrates how Power Query (M) can be used for splitting columns by a delimiter without query folding. It explains a typical scenario, key M functions used, and provides an example concept.

### Typical Scenario
In a real data pipeline, this transformation is useful when dealing with data sources that contain columns with delimited values that need to be split into separate columns for further analysis or processing.

### Core M Functions Used
`Table.SplitColumns`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"John;Smith;25"},
        {"Jane;Doe;30"}
    }, {"Name;Age"}),
    SplitName = Table.SplitColumns(Source, {"Name"}, {"Name.1"}, Splitter.SplitByDelimiter(";", QuoteStyle.None), ExtraValues.Error),
    RenamedColumns = Table.RenameColumns(SplitName, {{"Name.1", "First Name"}, {"Age", "Age"}})
    in
    RenamedColumns
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
1. The source table is created with a column 'Name;Age' containing delimited values.
2. The Table.SplitColumns function is used to split the 'Name' column by the ';' delimiter.
3. The result is a new table with the delimited values split into separate columns.
4. The columns are then renamed for clarity.
