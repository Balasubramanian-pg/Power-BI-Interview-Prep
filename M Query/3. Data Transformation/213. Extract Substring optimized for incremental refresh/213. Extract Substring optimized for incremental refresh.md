# Extract Substring optimized for incremental refresh

**Description:**
This transformation extracts a specific substring from a given text column, optimized for incremental refresh. It allows for efficient data processing by minimizing the amount of data that needs to be re-processed during refreshes. The use case is particularly useful when dealing with large datasets where only new or updated records need to be processed.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when ingesting log data from an application. For instance, extracting a specific error code or user ID from a log message can be achieved through substring extraction. This is particularly useful in incremental refresh scenarios where only new log entries are added, and the pipeline needs to efficiently process these new entries without re-processing the entire dataset.
