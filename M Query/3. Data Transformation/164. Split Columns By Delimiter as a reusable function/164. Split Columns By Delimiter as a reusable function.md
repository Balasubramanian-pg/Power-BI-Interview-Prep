# Split Columns By Delimiter as a Reusable Function

**Description:**
This transformation splits a column into multiple columns based on a specified delimiter. It is useful for data cleansing and normalization, allowing for more efficient data analysis and processing. By creating a reusable function, users can easily apply this transformation across multiple columns and tables.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when dealing with data imported from external sources, such as CSV files, where multiple values are stored in a single column separated by a delimiter. For instance, a column containing names and addresses separated by commas can be split into separate columns for names and addresses.
