# Parse Date From Text without query folding

**Description:**
This use case demonstrates how Power Query (M) can be used for parsing dates from text without query folding. It is useful for handling date data that is not in a standard format. The transformation helps in converting text representations of dates into a standard date format that can be used for analysis and processing.

### Typical Scenario
In a real data pipeline, this transformation would be used when ingesting data from various sources where date fields are represented in different text formats. For instance, when integrating data from multiple systems, some of which store dates as strings in non-standard formats (e.g., 'dd-mmm-yyyy' or 'mmm dd, yyyy').

### Core M Functions Used
`Date.Parse`, `Text.DateTime`

```powerquery
    Source = Table.FromRows({

    ParsedDate
```
