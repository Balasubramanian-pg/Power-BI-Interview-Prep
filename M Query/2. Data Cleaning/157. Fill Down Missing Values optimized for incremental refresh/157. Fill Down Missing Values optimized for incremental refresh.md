# Fill Down Missing Values optimized for incremental refresh

**Description:**
This transformation fills down missing values in a Power Query table, optimized for incremental refresh. It ensures that missing values are replaced with the last known value, making it suitable for datasets that are updated incrementally. This is particularly useful for maintaining data consistency over time. The transformation is efficient and can handle large datasets.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when dealing with time-series data that is updated incrementally. For instance, in financial data analysis, stock prices or trading volumes might be updated daily. If there are missing values in the dataset, this transformation ensures that the missing values are filled with the last known value, providing a complete and consistent dataset for analysis.

```powerquery
    PreviousRow = Table.AddColumn(Source, "Previous Value", each try List.Last(List.Skip(Source[Value], Table.RowCount(Source) - _ - 1)) otherwise null),
    FillDown = Table.ReplaceValue(PreviousRow, each [Value] = null, each [Previous Value], Replacer.ReplaceValue, {"Value"})
    in 
    FillDown
```

    3. The 'FillDown' step replaces the missing values in the 'Value' column with the corresponding 'Previous Value'.
    4. The resulting table has the missing values filled down with the last known value, making it suitable for incremental refresh.
