# Trim Text Columns for Large Datasets

**Description:**
This transformation removes leading and trailing whitespace from text columns in large datasets, ensuring data cleanliness and consistency. It is particularly useful for improving data quality and facilitating accurate analysis.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when ingesting data from various sources, such as CSV files or databases, where text columns may contain unnecessary whitespace. For instance, when loading customer data, trimming text columns like names and addresses helps prevent errors in downstream analysis and processing.
