# Change Column Types for Large Datasets

**Description:**
This transformation changes the data type of one or more columns in a large dataset. It is commonly used to ensure that data is correctly formatted for analysis or reporting. This transformation can be applied to a table with multiple columns and rows.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when ingesting data from a source system where the data types of certain columns are not correctly identified or need to be changed for further processing. For example, a column containing numeric data might be incorrectly imported as text and needs to be converted to a numeric type for calculations.


The transformation works as follows:
    1. **Source Table Creation**: A sample table named 'Source' is created with columns 'ID', 'Name', and 'Age'.
    2. **Type Transformation**: The Table.TransformColumns function is used to change the data type of the 'ID' and 'Age' columns from text to number.
    3. **Result**: The resulting table, 'ChangedType', contains the same data but with the 'ID' and 'Age' columns now correctly formatted as numbers.
