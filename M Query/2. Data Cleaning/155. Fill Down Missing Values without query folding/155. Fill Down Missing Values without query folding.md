# Fill Down Missing Values without query folding

**Description:**
This transformation fills down missing values in a Power Query table without relying on query folding. It ensures that null or missing values in a column are replaced with the last observed non-null value. This is particularly useful for handling incomplete data sets where values are missing but can be inferred from previous records.

### Typical Scenario
In a real data pipeline, this transformation is often used when dealing with time-series data or logs where some entries might be missing but can be assumed to have the same value as the previous entry. For instance, in financial transactions, if a transaction's currency is missing but the previous transaction had a specified currency, this transformation can fill in the missing currency.
