# Trim Text Columns as a reusable function

**Description:**
This transformation achieves the removal of leading and trailing whitespaces from text columns in a table, ensuring data cleanliness and consistency. It is particularly useful for standardizing data formats across different sources. By trimming text columns, data processing and analysis become more efficient and accurate.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when ingesting data from various sources, such as customer information, survey responses, or product descriptions. For instance, when collecting customer names or addresses, extra spaces might be inadvertently added during data entry. Trimming these text columns helps in maintaining a clean and standardized dataset, which is crucial for data analysis, reporting, and machine learning model training.

### Core M Functions Used
`Text.Trim`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {" John ", "New York"},
        {"Anna ", "Paris"},
        {"Peter ", "Berlin"}
    }, {"Name", "City"}),
    Trimmed = Table.TransformColumns(Source, {{"Name", each Text.Trim(_)}, {"City", each Text.Trim(_)}})
    in
    Trimmed
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
    1. **Define the Source Table**: A table with text columns is defined. In this example, the table has two columns: "Name" and "City".
    2. **Apply Text.Trim Function**: The `Table.TransformColumns` function is used to apply the `Text.Trim` function to each text column. The `Text.Trim` function removes leading and trailing whitespace from a text string.
    3. **Output the Trimmed Table**: The resulting table, with trimmed text columns, is outputted. This table can then be used for further data processing or analysis.
