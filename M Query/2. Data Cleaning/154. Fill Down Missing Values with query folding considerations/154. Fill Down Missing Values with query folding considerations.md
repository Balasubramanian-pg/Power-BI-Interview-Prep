# Fill Down Missing Values with Query Folding Considerations

**Description:**
This transformation fills down missing values in a Power Query table while considering query folding. It ensures that null or missing values in a column are replaced with the last non-null value. This is particularly useful for handling incomplete data sets where values are missing but can be inferred from previous records.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when dealing with data that is collected over time, such as sales data where some values might be missing for certain dates. For instance, if a sales report contains daily sales figures but some days are missing, filling down the missing values with the last known sales figure can provide a more complete picture of sales performance over time.
