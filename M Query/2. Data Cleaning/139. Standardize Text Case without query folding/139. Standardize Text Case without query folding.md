# Standardize Text Case without query folding

**Description:**
This transformation standardizes the text case in a dataset without relying on query folding. It ensures consistency in text data, which is crucial for accurate analysis and reporting. By standardizing text case, you can improve data quality and make it easier to search, filter, and analyze.

### Typical Scenario
In a real-world data pipeline, standardizing text case is often necessary when dealing with user-generated content, such as customer feedback or survey responses. For instance, a company may want to analyze customer feedback forms where the text data is entered in various cases (e.g., uppercase, lowercase, or mixed case). By standardizing the text case, the company can more effectively categorize and analyze the feedback.
