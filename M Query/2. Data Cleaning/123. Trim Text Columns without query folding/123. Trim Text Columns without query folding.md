# Trim Text Columns without query folding

**Description:**
This use case demonstrates how Power Query (M) can be used to trim text columns without query folding, which is useful for cleaning and preprocessing text data.

### Typical Scenario
In a real data pipeline, this transformation would be used when ingesting data from various sources, such as CSV files or databases, where text columns may contain leading or trailing whitespace that needs to be removed for accurate analysis or processing.

### Core M Functions Used
`Text.Trim`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"  John", " 25"},
        {"Alice  ", " 30"},
        {"Bob  ", " 35"}
    }, {"Name", "Age"}),
    TrimmedName = Table.TransformColumns(Source, {{"Name", each Text.Trim(_)}})
    in
    TrimmedName
```

### Example Execution Logic Step-By-Step
The transformation works by first creating a table with sample data. Then, it uses the Table.TransformColumns function to apply the Text.Trim function to each value in the 'Name' column. The Text.Trim function removes any leading or trailing whitespace from the text values, resulting in a cleaned table with trimmed names.
