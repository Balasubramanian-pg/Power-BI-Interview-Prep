# Replace Null Values without Query Folding

**Description:**
This transformation replaces null values in a table without relying on query folding. It is useful for handling null values when the data source does not support query folding. By using Power Query (M), you can efficiently replace null values in your data.

### Typical Scenario
In a real data pipeline, this transformation is used when dealing with data sources that do not support query folding. For instance, when loading data from a source that does not optimize queries, replacing null values directly in Power Query ensures data integrity and consistency.
