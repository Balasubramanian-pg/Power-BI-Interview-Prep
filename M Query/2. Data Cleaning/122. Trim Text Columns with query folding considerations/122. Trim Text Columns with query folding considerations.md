# Trim Text Columns with Query Folding Considerations

**Description:**
This transformation trims text columns in a table, removing leading and trailing whitespace. It is useful for data cleansing and normalization. Query folding considerations ensure efficient data processing.

### Typical Scenario
In a real data pipeline, this transformation is used when ingesting data from a source where text columns may contain extra whitespace, such as when reading data from a CSV file or a database. For instance, customer names or addresses may have leading or trailing spaces that need to be cleaned up for accurate analysis.
