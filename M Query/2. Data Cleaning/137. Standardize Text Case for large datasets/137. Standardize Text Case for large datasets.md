# Standardize Text Case for Large Datasets

**Description:**
This transformation standardizes the text case in large datasets, ensuring consistency across the data. It is particularly useful for cleaning and preprocessing data for analysis or reporting. By standardizing text case, it becomes easier to compare, categorize, and analyze text data.

### Typical Scenario
In a real-world data pipeline, standardizing text case is crucial when integrating data from multiple sources, such as customer information from different CRM systems. For instance, when merging customer data, standardizing the case of city names, country names, or company names ensures that 'New York', 'NEW YORK', and 'new york' are treated as the same value, facilitating accurate analysis and reporting.

`Text.Upper`, `Text.Lower`, `Text.Proper`
