# Clean Invalid Characters as a reusable function

**Description:**
This transformation achieves the removal of invalid characters from a dataset, making it cleaner and more reliable for analysis. It helps in standardizing the data by eliminating unwanted characters that could interfere with data processing or analysis. This is particularly useful in data integration and data quality improvement processes.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when ingesting data from various sources that may contain invalid or inconsistent characters. For instance, when loading customer data from a CRM system, the data might include special characters or non-standard text that needs to be cleaned before being loaded into a data warehouse for analysis.

The transformation works by defining a custom function `CleanInvalidChars` that takes a text input and replaces specified invalid characters with an empty string. This function is then applied to the 'Email' column of the `Source` table using `Table.TransformColumns`. The result is a cleaned dataset where specified invalid characters have been removed from the email addresses.
