# Trim Text Columns optimized for incremental refresh

**Description:**
This transformation trims text columns to remove leading and trailing whitespaces, ensuring data cleanliness and consistency. It is particularly useful when working with data that may have inconsistent spacing. By optimizing for incremental refresh, it allows for efficient updates to the data without requiring a full reload.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial when integrating data from multiple sources, such as customer information from CRM systems. For instance, customer names might be entered with varying amounts of whitespace, leading to inconsistencies in reporting and analysis. By trimming these text columns during the data ingestion process, businesses can maintain a clean and standardized dataset, which improves data quality and reliability.

### Core M Functions Used
`Table.TransformColumns`, `Text.Trim`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({

        {" Peter ", " 35 "}
    }, {"Name", "Age"}),
    TrimmedColumns = Table.TransformColumns(Source, {{"Name", each Text.Trim(_)}})
    in
        TrimmedColumns
```

The transformation works as follows:
    1. **Source Table Creation**: A table is created with sample data, including names with leading and trailing whitespaces.
    2. **Transformation Application**: The `Table.TransformColumns` function is applied to the source table, specifically targeting the 'Name' column.
    3. **Text Trimming**: Within the transformation, the `Text.Trim` function is used to remove leading and trailing whitespaces from each value in the 'Name' column.
    4. **Result**: The resulting table contains the same data but with the names cleaned up, ensuring that there are no leading or trailing whitespaces in the 'Name' column.
