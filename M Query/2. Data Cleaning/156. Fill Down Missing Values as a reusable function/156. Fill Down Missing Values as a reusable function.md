# Fill Down Missing Values as a reusable function

**Description:**
This transformation achieves filling down missing values in a Power Query table. It allows for a reusable function to be created, making it easier to apply this transformation across multiple tables or queries. By filling down missing values, you can ensure data consistency and accuracy.

### Typical Scenario
In a real data pipeline, this transformation would be used when dealing with datasets that have missing values in certain columns. For instance, when processing sales data, some sales records might not have a region specified, but the region information is available for the preceding record. By filling down the missing values, you can ensure that each sales record has a region associated with it, making it easier to analyze and report on the data.
