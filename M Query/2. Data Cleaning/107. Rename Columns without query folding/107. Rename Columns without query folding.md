# Rename Columns without Query Folding

**Description:**
This transformation allows for renaming columns in a Power Query table without relying on query folding. It is useful when dealing with data sources that do not support query folding or when complex transformations need to be applied.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when ingesting data from a source that doesn't support query folding, such as certain web APIs or data stored in a file system. For instance, when data is extracted from an API and needs to be transformed before loading into a data warehouse.
