# Turn Wide Data Into Musical Notes Optimized for Incremental Refresh

**Description:**
This transformation takes wide data and converts it into musical notes, optimizing it for incremental refresh. It enables efficient data processing and analysis by restructuring the data into a suitable format for musical note representation. This use case demonstrates the application of Power Query (M) for such transformations.

### Typical Scenario
In a music streaming service, data is collected on user interactions with various songs, including ratings and play counts. This data is initially in a wide format with columns for each type of interaction. To analyze and generate musical recommendations based on user preferences, the data needs to be transformed into a long format where each row represents a musical note or an interaction event. This transformation is crucial for incremental refresh, allowing the service to continuously update user preferences and song recommendations in real-time.
