# Obfuscate And De-obfuscate Data as a reusable function

**Description:**
This use case demonstrates how Power Query (M) can be used for obfuscating and de-obfuscating data as a reusable function. It provides a way to protect sensitive information in data pipelines.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when sensitive data, such as personal identifiable information (PII) or confidential business data, needs to be protected during data processing and storage. For example, in a data warehousing scenario, customer names and addresses might need to be obfuscated for analytics purposes.

### Code Source Execution
```powerquery
let
    Obfuscate = (data as text) =>
        let
            obfuscatedData = Text.Combine(List.Transform(Text.ToList(data), each "*"), ""),
            result = obfuscatedData
        in
            result,
    Deobfuscate = (obfuscatedData as text, originalData as text) =>
        let
            deobfuscatedData = originalData,
            result = deobfuscatedData
        in
            result,
    data = {"John Doe", "Jane Doe"},
    obfuscatedData = List.Transform(data, Obfuscate)
    in
        obfuscatedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
    1. The Obfuscate function takes a text input and replaces each character with an asterisk (*).
    2. The Deobfuscate function takes an obfuscated text and the original text, and returns the original text.
    3. In a Power Query pipeline, you would apply the Obfuscate function to sensitive data to protect it.
    4. When the original data is needed, you would apply the Deobfuscate function to retrieve it.
