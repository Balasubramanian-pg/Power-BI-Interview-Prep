# Self-documenting Query Generator for large datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to generate self-documenting queries for large datasets. It allows for the creation of queries that can handle large volumes of data while also providing clear documentation of the data transformation process.

### Typical Scenario
In a real data pipeline, this transformation would be used when dealing with large datasets that require complex data transformations. For instance, when integrating data from multiple sources, a self-documenting query generator can help automate the data transformation process while maintaining clear documentation of the transformations applied.
