# Transforming Wide Data into Musical Notes without Query Folding

**Description:**
This transformation takes wide data and converts it into musical notes without relying on query folding. It is particularly useful in scenarios where data needs to be creatively repurposed. The process involves selecting specific rows and transforming them into a musical note format.

### Typical Scenario
In a music production data pipeline, there might be a need to convert wide data tables containing note information into a format that can be read by music synthesis software. For instance, a table with columns for note names, durations, and frequencies needs to be transformed into a sequence of musical notes that can be played back.

`Table.SelectRows`, `List.Transform`, `Text.Combine`

    CombinedNotes = Text.Combine(
        Table.ToList(NoteFormat),

    )
    in
        CombinedNotes
