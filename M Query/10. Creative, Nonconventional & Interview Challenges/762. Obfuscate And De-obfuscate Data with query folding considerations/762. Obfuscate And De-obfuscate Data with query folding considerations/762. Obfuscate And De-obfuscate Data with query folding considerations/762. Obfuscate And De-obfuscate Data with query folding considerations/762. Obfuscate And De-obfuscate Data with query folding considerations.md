# Obfuscate And De-obfuscate Data with query folding considerations

**Description:**
This use case demonstrates how Power Query (M) can be used for obfuscating and de-obfuscating data while considering query folding. It helps protect sensitive information during data processing and analysis.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when dealing with sensitive customer information, such as credit card numbers or personal identifiable information (PII). For instance, a company may want to obfuscate credit card numbers during data processing to prevent unauthorized access, and then de-obfuscate them only for specific business operations.

### Core M Functions Used
`Table.SelectRows`, `List.Transform`, `Text.Replace`

### Code Source Execution
```powerquery
let
    // Sample table with sensitive data
    data = Table.FromList({
        {"John", "1234-5678-9012-3456"},
        {"Jane", "9876-5432-1098-7654"}
    }, Splitter.SplitByNothing(), {"Name", "Credit Card"}),
    // Obfuscate credit card numbers
    obfuscatedData = Table.TransformColumns(data, {{"Credit Card", each Text.Replace(_, "[0-9]", "*")}}),
    // De-obfuscate credit card numbers (for demonstration purposes only)
    deobfuscatedData = Table.TransformColumns(obfuscatedData, {{"Credit Card", each Text.Replace(_, "*", "0")}})
    in
    deobfuscatedData
```

The transformation works as follows:
    1. Start with a sample table containing sensitive data, such as credit card numbers.

    3. The obfuscated data can be processed and analyzed without exposing sensitive information.
    4. When needed, de-obfuscate the credit card numbers by replacing the asterisks with the original digits (for demonstration purposes only).
