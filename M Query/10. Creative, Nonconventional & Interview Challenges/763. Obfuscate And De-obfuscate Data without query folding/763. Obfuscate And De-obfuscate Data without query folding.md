# Obfuscate And De-obfuscate Data without query folding

**Description:**
This use case demonstrates how Power Query (M) can be used for obfuscating and de-obfuscating data without query folding, ensuring data protection and security in data pipelines.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when sensitive data needs to be protected during data processing or transmission. For instance, when extracting customer information from a database, the email addresses and phone numbers can be obfuscated to prevent unauthorized access.

### Core M Functions Used
`Text.Replace`, `Text.Reverse`

### Code Source Execution
```powerquery
let
    // Sample data
    data = #"2023-01-01" [ Name = "John", Email = "john@example.com" ],
    // Obfuscate email
    obfuscateEmail = (email) => Text.Replace(email, "@", " at "),
    // De-obfuscate email
    deobfuscateEmail = (email) => Text.Replace(email, " at ", "@"),
    // Apply obfuscation and de-obfuscation
    obfuscatedData = Table.TransformColumns(data, {{"Email", obfuscateEmail}}),
    deobfuscatedData = Table.TransformColumns(obfuscatedData, {{"Email", deobfuscateEmail}})
    in
    deobfuscatedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
1. Start with a table containing sensitive data.
2. Apply an obfuscation function to the sensitive columns (e.g., replacing '@' with ' at ' in email addresses).
3. Process the obfuscated data as needed.
4. Apply a de-obfuscation function to restore the original data (e.g., replacing ' at ' back with '@').
