# Transforming Wide Data into Musical Notes using Power Query

**Description:**
This transformation takes wide data and converts it into musical notes, providing a reusable function for music-related data processing. It enables users to easily transform data into a musical format.

### Typical Scenario
In a music production pipeline, this transformation can be used to convert tabular data containing note information into a format that can be used by music software, such as note duration, pitch, and instrument.
