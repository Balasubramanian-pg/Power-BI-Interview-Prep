# Create A Procedural Fake Dataset for multi-tenant scenarios

**Description:**
This use case demonstrates how Power Query (M) can be used to generate a procedural fake dataset for multi-tenant scenarios, allowing for the creation of realistic and varied data for testing and development purposes.

### Typical Scenario
In a real data pipeline, this transformation would be used when a development team needs to test a multi-tenant application or data model with realistic data. For instance, a company building a SaaS product might use this approach to generate fake customer data for testing and quality assurance.

### Core M Functions Used
`Table.SelectRows`, `List.Generate`, `Text.From`

### Code Source Execution
```powerquery
let
    // Define the number of tenants and records per tenant
    numberOfTenants = 5,
    recordsPerTenant = 100,

    // Generate a list of tenant IDs
    tenantIds = List.Generate(
        () => 1,
        each _ < numberOfTenants + 1,
        each _ + 1
    ),

    // Generate fake data for each tenant
    data = List.Transform(tenantIds, each
        Table.FromRows(
            List.Generate(
                () => [#"Tenant ID" = _, #"Record ID" = 1],
                each [#"Record ID" = _["Record ID"] + 1],
                each if _["Record ID"] <= recordsPerTenant then [#"Tenant ID" = _["Tenant ID"], #"Record ID" = _["Record ID"]] else null,
                {"Tenant ID", "Record ID"}
            )
        )
    ),

    // Combine the data from all tenants into a single table
    combinedData = Table.Combine(data)

in
    combinedData
```

### Example Execution Logic Step-By-Step
This script works as follows:
1. It first defines the number of tenants and the number of records per tenant.
2. It generates a list of tenant IDs using `List.Generate`.
3. For each tenant ID, it generates a table of fake data using `List.Transform` and `Table.FromRows`.
4. The fake data includes a tenant ID and a record ID.
5. Finally, it combines all the data from the different tenants into a single table using `Table.Combine`.
