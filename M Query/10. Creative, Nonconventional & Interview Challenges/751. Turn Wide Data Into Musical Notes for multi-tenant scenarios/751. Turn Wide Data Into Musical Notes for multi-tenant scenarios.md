# Transforming Wide Data into Musical Notes for Multi-Tenant Scenarios

**Description:**
This transformation takes wide data and converts it into musical notes, enabling multi-tenant scenarios. It achieves this by pivoting and transforming data into a suitable format for musical note representation.

### Typical Scenario
In a music streaming service with multiple tenants, each having their own set of songs with wide data (e.g., song attributes like tempo, pitch, etc.), this transformation can be used to convert the wide data into musical notes that can be used for analysis or processing.

### Core M Functions Used
`Table.SelectRows`, `Table.Pivot`, `List.Distinct`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"Tenant1", "Song1", 100, 200, 300},
        {"Tenant1", "Song2", 400, 500, 600},
        {"Tenant2", "Song3", 700, 800, 900}
    }, {"Tenant", "Song", "Pitch", "Tempo", "Duration"}),
    SelectTenantData = Table.SelectRows(Source, each [Tenant] = "Tenant1"),
    PivotData = Table.Pivot(SelectTenantData, List.Distinct(SelectTenantData[Song]), "Song"),
    TransformToNotes = (Pitch, Tempo, Duration) => {
        "Note": Pitch * Tempo * Duration
    },
    AddNotesColumn = Table.AddColumn(PivotData, "Notes", each TransformToNotes([Pitch], [Tempo], [Duration]))
in AddNotesColumn
```
