# Combine All Files In Folder without query folding

**Description:**
This use case demonstrates how Power Query (M) can be used to combine all files in a folder without query folding, which is useful for handling files that do not support query folding.

### Typical Scenario
In a real data pipeline, this transformation would be used when dealing with a folder containing multiple files of the same structure, such as CSV or Excel files, that need to be combined into a single dataset for analysis or reporting.

### Core M Functions Used
`Folder.Files`, `Table.Combine`

### Code Source Execution
```powerquery
let
    Source = Folder.Files("C:\Path\To\Folder"),
    CombinedData = Table.Combine(Source, "Data"),
    PromotedHeaders = Table.PromoteHeaders(CombinedData, [PromoteAllScalars=true])
    in
        PromotedHeaders
```

### Example Execution Logic Step-By-Step
The transformation works by first using the Folder.Files function to get a list of files in the specified folder. Then, it uses Table.Combine to combine the data from each file into a single table. Finally, it promotes the headers of the combined table using Table.PromoteHeaders.
