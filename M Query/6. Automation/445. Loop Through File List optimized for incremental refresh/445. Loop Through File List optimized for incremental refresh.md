# Loop Through File List Optimized for Incremental Refresh

**Description:**
This use case demonstrates how Power Query (M) can be used to loop through a file list in a way that is optimized for incremental refresh. It allows for efficient data loading by processing only new or updated files since the last refresh.

### Typical Scenario
In a real data pipeline, this transformation would be used when you have a folder containing multiple files (e.g., CSV, Excel) that are updated periodically. For instance, a business receives daily sales reports from different regions, and each report is saved as a separate file in a folder. By looping through the file list optimized for incremental refresh, the business can automate the data ingestion process, ensuring that only new or updated files are processed, thus saving time and resources.

### Core M Functions Used
`File.Contents`, `Folder.Files`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    // Specify the folder path
    folderPath = "C:\Path\To\Files",

    // Get the list of files in the folder
    files = Folder.Files(folderPath),

    // Filter files by type (e.g., .csv)
    csvFiles = Table.SelectRows(files, each [Kind] = "File" and [Extension] = ".csv"),

    // Loop through each file and load its content
    data = Table.AddColumn(csvFiles, "Data", each
        let
            fileContent = File.Contents([AbsolutePath]),
            data = Csv.Document(fileContent, [Delimiter=",", Encoding=1252])
        in
            data
    )
in
    data
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
1. **Specify the Folder Path**: Define the path to the folder containing the files you want to process.
2. **Get the List of Files**: Use `Folder.Files` to get a table of files in the specified folder.
3. **Filter Files**: Use `Table.SelectRows` to filter the files based on their type (e.g., only CSV files).
4. **Loop Through Each File**: Add a custom column to the table of files where each file's content is loaded using `File.Contents` and then parsed using `Csv.Document`.
5. **Incremental Refresh**: To optimize for incremental refresh, you would typically track the last modified date of the files and only process files that have changed since the last refresh.
