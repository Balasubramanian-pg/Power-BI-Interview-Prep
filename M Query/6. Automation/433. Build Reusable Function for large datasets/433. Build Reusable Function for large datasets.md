# Build Reusable Function for Large Datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to build reusable functions for handling large datasets. It enables data transformation and processing in a scalable and efficient manner.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when dealing with large datasets that require complex data processing and transformation. For instance, when ingesting data from multiple sources, performing data cleansing, and applying business logic to derive new columns or filter data.

### Core M Functions Used
`Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"John", 25, "USA"},
        {"Mary", 31, "Canada"},
        {"Peter", 42, "UK"}
    }, {"Name", "Age", "Country"}),
    FilteredRows = Table.SelectRows(Source, each [Age] > 30)
    in
    FilteredRows
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
1. A sample table named 'Source' is created with columns 'Name', 'Age', and 'Country'.
2. The Table.SelectRows function is used to filter rows from the 'Source' table where the 'Age' is greater than 30.
3. The result is a new table, 'FilteredRows', containing only the rows that meet the specified condition.
