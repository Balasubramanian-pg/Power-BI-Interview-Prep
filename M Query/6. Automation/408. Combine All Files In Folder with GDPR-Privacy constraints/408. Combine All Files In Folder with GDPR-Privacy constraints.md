# Combine All Files In Folder with GDPR/Privacy constraints

**Description:**
This use case demonstrates how Power Query (M) can be used to combine all files in a folder while adhering to GDPR/privacy constraints. It ensures that sensitive data is handled appropriately during the data transformation process.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when a company needs to aggregate data from multiple files stored in a folder, while ensuring that the data processing complies with GDPR and other privacy regulations. For instance, a company might need to combine monthly sales data from various regional offices, where each office's data is stored in a separate file.
