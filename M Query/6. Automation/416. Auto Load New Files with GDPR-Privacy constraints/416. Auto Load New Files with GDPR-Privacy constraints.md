# Auto Load New Files with GDPR/Privacy Constraints

**Description:**
This use case demonstrates how Power Query (M) can be used to automatically load new files while adhering to GDPR and privacy constraints. It ensures that sensitive data is handled appropriately and in compliance with regulations.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when a company needs to regularly ingest new data from files, such as daily or weekly reports, while ensuring that the data processing complies with GDPR and other privacy laws. For instance, a financial institution might use this to load transactional data from new files each day, ensuring that personally identifiable information is properly anonymized or handled.

### Core M Functions Used
`Table.SelectRows`, `Folder.Files`, `File.Contents`

### Code Source Execution
```powerquery

    Source = Folder.Files("C:\Path\To\Files"),
    FilesToLoad = Table.SelectRows(Source, each [Extension] = ".csv"),

    in
    LoadedFiles
```
