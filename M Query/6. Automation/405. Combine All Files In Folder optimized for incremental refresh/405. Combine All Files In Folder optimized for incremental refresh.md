# Combine All Files In Folder optimized for incremental refresh

**Description:**
This use case demonstrates how Power Query (M) can be used to combine all files in a folder, optimized for incremental refresh. It allows for efficient data loading and updating by only loading new or changed files.

### Typical Scenario
In a real data pipeline, this transformation would be used when there is a need to aggregate data from multiple files stored in a folder, and the data is constantly being updated. For example, a company receives daily sales reports in CSV format from different regions, and these reports are stored in a folder. The company wants to combine these reports into a single dataset for analysis, but only wants to load new or updated reports each day.

### Core M Functions Used
`Folder.Files`, `Table.Combine`

### Code Source Execution
```powerquery
let
    Source = Folder.Files("C:\Path\To\Folder"),
    CombinedData = Table.Combine(
        Table.SelectRows(Source, each [Kind] = "File"),
        "Content"
    )
    in
    CombinedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
1. The `Folder.Files` function is used to get a list of files in the specified folder. 
2. The `Table.SelectRows` function is used to filter the list and only select files (not folders). 
3. The `Table.Combine` function is used to combine the contents of all the files into a single table. 
4. For incremental refresh, a date column can be added to track when each file was last updated, and only files with a newer date can be loaded.
