# Combine All Files In Folder as a Reusable Function

**Description:**
This use case demonstrates how Power Query (M) can be used to combine all files in a folder into a single table. It provides a reusable function to achieve this, making it easier to work with multiple files in a data pipeline.

### Typical Scenario
In a real-world data pipeline, this transformation is useful when dealing with a large number of files in a folder, such as monthly or quarterly reports, that need to be combined into a single dataset for analysis.

### Core M Functions Used
`Folder.Files`, `Table.Combine`

### Code Source Execution
```powerquery
let
    Source = Folder.Files("C:\Path\To\Folder"),
    CombinedData = Table.Combine(Source, "Data"),
    PromotedHeaders = Table.PromoteHeaders(CombinedData, [PromoteAllScalars=true])
in PromotedHeaders
```

### Example Execution Logic Step-By-Step
The script works as follows:
1. The `Folder.Files` function is used to get a list of files in the specified folder.
2. The `Table.Combine` function is then used to combine the data from all files into a single table.
3. Finally, the `Table.PromoteHeaders` function is used to promote the first row of the combined table to headers.
