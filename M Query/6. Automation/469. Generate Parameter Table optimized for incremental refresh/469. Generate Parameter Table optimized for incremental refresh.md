# Generate Parameter Table optimized for incremental refresh

**Description:**
This transformation generates a parameter table optimized for incremental refresh in Power Query (M). It enables efficient data loading by creating a table with parameters that facilitate incremental data refresh. This approach helps reduce data volume and improve performance.

### Typical Scenario
In a real-world data pipeline, this transformation is used when dealing with large datasets that require frequent updates. For instance, in a sales data pipeline, a parameter table can be generated to track changes in sales data since the last refresh, allowing for incremental updates rather than reloading the entire dataset.

`Table.SelectRows`, `Table.FromList`, `List.Dates`

The transformation works as follows:
1. Define the start and end dates for the parameter table.

3. Create a table from the list of dates using Table.FromList.
4. Transform the column type of the date table to a date type.
5. Use Table.SelectRows to filter the date table and create the parameter table with dates within the specified range.

This approach allows for efficient generation of a parameter table optimized for incremental refresh.
