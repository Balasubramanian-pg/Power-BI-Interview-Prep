# Auto Load New Files as a Reusable Function

**Description:**
This use case demonstrates how Power Query (M) can be used to automatically load new files as a reusable function. It allows for dynamic data ingestion from newly added files without manual intervention.

### Typical Scenario
In a real-world data pipeline, this transformation is useful when data is constantly being updated with new files. For instance, a company receives daily sales reports in CSV format. The Auto Load New Files function can be used to automatically ingest these new files into the data pipeline, ensuring that the data is always up-to-date.

### Core M Functions Used
`Folder.Files`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Folder.Files("C:\Path\To\Folder"),
    FilesToLoad = Table.SelectRows(Source, each [Extension] = ".csv"),
    CombinedData = Table.Combine(FilesToLoad, "Data")
    in
    CombinedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
1. **Folder.Files**: This function is used to get a list of files from a specified folder.
2. **Table.SelectRows**: This function filters the list of files to only include those with a specific extension (e.g., .csv).
3. **Table.Combine**: This function combines the data from the selected files into a single table.

By wrapping these steps in a reusable function, you can easily automate the process of loading new files as they become available.
