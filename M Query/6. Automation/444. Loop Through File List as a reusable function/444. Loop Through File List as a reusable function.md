# Loop Through File List as a Reusable Function

**Description:**
This transformation allows you to iterate over a list of files and perform operations on each file. It is useful when you need to process multiple files in a data pipeline. The transformation can be used to read, transform, and combine data from multiple files.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when you receive daily sales reports from different regions in CSV format. You can use this transformation to loop through the list of files, read each file, and combine the data into a single table for analysis.

### Core M Functions Used
`Table.SelectRows`, `List.Transform`, `File.Contents`

```powerquery
let
    // Assume 'files' is a list of file paths
    files = {"file1.csv", "file2.csv"},
    // Define a function to process each file
    ProcessFile = (filePath) =>
        let
            // Read the file contents
            fileContents = File.Contents(filePath),
            // Transform the file contents into a table
            table = Csv.Document(fileContents, [Delimiter=",", Encoding=1252]),
            // Select specific columns
            selectedColumns = Table.SelectRows(table, "Column1 & Column2")
        in
            selectedColumns,
    // Loop through the list of files and apply the function
    results = List.Transform(files, ProcessFile)
    // Combine the results into a single table
    in Table.Combine(results)
```
