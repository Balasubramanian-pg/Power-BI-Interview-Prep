# Loop Through File List for Large Datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to loop through a list of files for large datasets, allowing for efficient data processing and transformation.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when dealing with large datasets spread across multiple files, such as log files, CSVs, or Excel files, where each file needs to be processed individually.

### Core M Functions Used
`Folder.Files`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    // Get list of files
    files = Folder.Files("C:\Path\To\Files"),
    // Loop through each file and combine data
    combinedData = Table.CombineRows(
        Table.SelectRows(files, "= each [Kind] = \"File\""),
        "Combine"
    )
    // Perform further transformations
in
    combinedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
1. First, the `Folder.Files` function is used to get a list of files from a specified directory.
2. Then, `Table.SelectRows` is used to filter the list to only include files (not folders).
3. The `Table.CombineRows` function is then used to combine the data from each file into a single table.
4. Finally, further transformations can be performed on the combined data.
