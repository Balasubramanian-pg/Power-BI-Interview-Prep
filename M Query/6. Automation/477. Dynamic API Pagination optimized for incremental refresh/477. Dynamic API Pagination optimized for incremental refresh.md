# Dynamic API Pagination for Incremental Refresh

**Description:**
This transformation optimizes data retrieval from APIs with dynamic pagination, enabling efficient incremental data refreshes. It allows for real-time data updates by fetching only new or updated records.

### Typical Scenario
In a data pipeline that fetches customer data from a CRM API, which uses cursor-based pagination. The pipeline needs to fetch new customers or updated customer information daily without duplicates.

### Core M Functions Used
`Web.Contents`, `Json.Document`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    url = "https://api.example.com/customers",
    params = ["pageSize" = 100, "cursor" = null],
    GetJson = (url, params) =>
        Web.Contents(url, [Query = params]),
    JsonToTable = (json) =>
        Json.Document(json),
    GetCustomers = (cursor) =>
        let
            params = if cursor = null then ["pageSize" = 100] else ["pageSize" = 100, "cursor" = cursor],
            json = GetJson(url, params),
            table = JsonToTable(json)
        in
            table,
    customers = GetCustomers(null)
    in
        customers
```
