# Auto Load New Files with Error Handling and Retries

**Description:**
This use case demonstrates how Power Query (M) can be used to automatically load new files with robust error handling and retry mechanisms. It ensures that data ingestion is reliable and resilient, handling potential issues such as file not found errors or network connectivity problems.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial for scenarios where new files are continuously being added to a directory, such as in IoT data ingestion or log file collection. For instance, a company might have a folder where new sales data CSV files are dropped every hour. The pipeline needs to automatically pick up these new files, handle any errors that might occur during file access or data conversion, and retry failed operations to ensure data consistency.

### Core M Functions Used
`File.Contents`, `Folder.Files`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    // Source folder path
    sourceFolder = "C:\Source\NewFiles",

    // Get list of files in the folder
    files = Folder.Files(sourceFolder),

    // Filter for new files (example: .csv files)
    newFiles = Table.SelectRows(files, each [Kind] = "File" and [Extension] = ".csv"),

    // Function to load file contents with error handling
    loadFile = (filePath) =>
        try
            File.Contents(filePath)
        catch
            [Error] =>
                // Retry mechanism (example: 3 retries)
                if [Error][Message] = "File not found" then
                    try
                        File.Contents(filePath)
                    catch
                        [Error] =>
                            try
                                File.Contents(filePath)
                            catch
                                [Error] =>
                                    Error.Record("File failed to load after retries", [Error][Message], [Error][StackTrace])
                else
                    Error.Record("An error occurred", [Error][Message], [Error][StackTrace]),

    // Load contents of new files
    loadedFiles = Table.AddColumn(newFiles, "File Contents", each loadFile([FilePath]))

in
    loadedFiles
```
