# Combine All Files In Folder for Large Datasets

**Description:**
This transformation combines data from all files within a specified folder, ideal for handling large datasets by aggregating data from multiple files into a single dataset.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial when dealing with large datasets spread across multiple files, such as monthly sales reports or daily transaction logs. For instance, a company might have a folder containing monthly sales reports in CSV format for the past five years. By using this transformation, the company can easily aggregate all these reports into a single dataset for analysis or reporting purposes.

### Core M Functions Used
`Folder.Files`, `File.Contents`, `Table.Combine`

### Code Source Execution
```powerquery
let
    Source = Folder.Files("C:\Path\To\Folder"),
    CombinedData = Table.Combine(
        List.Transform(Source, each Table.FromRows(File.Contents([Content])))
    )
    in
    CombinedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
1. **Folder.Files**: This function is used to get a list of files in the specified folder. It returns a table with file metadata.
2. **List.Transform**: This function transforms each file in the list by reading its content using **File.Contents**.
3. **Table.FromRows**: Each file content is then converted into a table.
4. **Table.Combine**: Finally, all the tables (each representing a file) are combined into a single table.

This step-by-step process allows for the aggregation of data from multiple files into a single dataset, which can then be further processed or analyzed.
