# Auto Load New Files with Query Folding Considerations

**Description:**
This use case demonstrates how Power Query (M) can be used to automatically load new files while considering query folding. It allows for efficient data loading and updating of data pipelines. The transformation enables users to dynamically add new files to their data pipeline.

### Typical Scenario
In a real data pipeline, this transformation would be used when new files are constantly being added to a directory, such as daily or weekly reports. For instance, a company receives daily sales reports in CSV format, and these reports need to be automatically loaded into their data warehouse for analysis.

### Core M Functions Used
`Folder.Files`, `Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Folder.Files("C:\Path\To\Files"),
    CombinedRows = Table.CombineRows(Source, "Combine Files.Contents", {"Name", "Extension"}, "[Content]"),
    SelectedRows = Table.SelectRows(CombinedRows, each [Extension] = ".csv")
    in
    SelectedRows
```

### Example Execution Logic Step-By-Step
The transformation works as follows: 
    1. The `Folder.Files` function is used to get a list of files in a specified directory.
    2. The `Table.CombineRows` function is then used to combine the contents of these files into a single table.
    3. The `Table.SelectRows` function is used to filter the files based on their extensions, for example, selecting only .csv files.
    4. The resulting table contains the contents of all the selected files, which can then be loaded into the data pipeline.
