# Dynamic Date Filter for Large Datasets

**Description:**
This transformation enables efficient filtering of large datasets based on dynamic date criteria. It allows users to easily apply date filters without manual intervention, making it ideal for automated data pipelines. By leveraging Power Query's M language, this solution provides a flexible and scalable approach to date-based data filtering.

### Typical Scenario
In a real-world data pipeline, this transformation is useful when dealing with large datasets that need to be filtered based on dynamic date ranges, such as extracting data for the last 30 days, a specific quarter, or a custom date range provided by users.

`Table.SelectRows`

```powerquery
let

        {"2022-01-01", "John", 100},

    DynamicDateFilter = Table.SelectRows(Source, each [Date] >= Date.AddDays(DateTime.Date(DateTime.Now()), -30))
    in
    DynamicDateFilter
```

    3. **Apply Filter**: Use `Table.SelectRows` to filter the table. This function takes a table and a condition as arguments. The condition is specified using the `each` keyword, allowing you to define a criteria for each row.
    4. **Result**: The resulting table will contain only the rows where the date falls within the specified dynamic range.
