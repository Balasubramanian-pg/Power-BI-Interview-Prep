# Schedule Data Refresh without query folding

**Description:**
This use case demonstrates how Power Query (M) can be used for scheduling data refresh without query folding. It allows for refreshing data at regular intervals.

### Typical Scenario
In a real data pipeline, this transformation would be used when there is a need to periodically refresh data from a source that does not support query folding, such as a database or a web API.

### Core M Functions Used
`Table.SelectRows`

### Code Source Execution
```powerquery
let
    Source = Table.FromRows({
        {"ID", "Name", "Age"},
        {1, "John", 25},
        {2, "Jane", 30}
    }),
    Refresh = () =>
        Table.SelectRows(Source, each true)
    in
        Refresh
```

### Example Execution Logic Step-By-Step
The transformation works by first loading the data into a table. Then, it defines a function 'Refresh' that selects all rows from the table. This function can be called at regular intervals to refresh the data.
