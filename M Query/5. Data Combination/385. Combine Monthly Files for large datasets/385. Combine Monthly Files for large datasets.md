# Combine Monthly Files for Large Datasets

**Description:**
This transformation combines multiple monthly files into a single dataset, enabling efficient analysis and processing of large datasets. It is particularly useful for handling time-series data that is split across multiple files.

### Typical Scenario
In a real-world data pipeline, this transformation would be used when a business needs to analyze sales data from multiple months stored in separate files. For instance, a company might have a file for each month's sales data, and they want to combine these files into a single dataset for analysis and reporting.

### Core M Functions Used
`Table.Combine`, `Folder.Files`, `File.Contents`

### Code Source Execution
```powerquery
let
    Source = Folder.Files("C:\Path\To\Files"),
    CombinedData = Table.Combine(
        Table.SelectRows(Source, each [Extension] = ".csv"),
        "Content"
    )
    in
    CombinedData
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
1. **Folder.Files**: This function is used to get a list of files from a specified folder.
2. **Table.SelectRows**: This function filters the list of files to only include CSV files.
3. **Table.Combine**: This function combines the contents of the filtered files into a single table.
The result is a single table containing data from all the monthly files.
