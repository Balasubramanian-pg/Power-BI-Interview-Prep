# Full Outer Join for large datasets

**Description:**
This use case demonstrates how Power Query (M) can be used to perform a full outer join on large datasets. A full outer join returns all records from both tables, with nulls in the columns where no match exists.

### Typical Scenario
In a real data pipeline, a full outer join might be used to combine customer information from two separate databases, ensuring that all customers are included, even if they don't have matching records in the other database.
