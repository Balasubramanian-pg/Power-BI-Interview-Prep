# Right Join Datasets with GDPR/Privacy constraints

**Description:**
This use case demonstrates how Power Query (M) can be used to right join datasets while adhering to GDPR/privacy constraints. It ensures that sensitive data is handled appropriately during the join operation.

### Typical Scenario
In a real-world data pipeline, this transformation is crucial when combining customer data from different sources while ensuring compliance with GDPR. For instance, joining customer information from a CRM system with order data from an e-commerce platform, all while masking or anonymizing sensitive personal data.
