**Description:**

### Typical Scenario
In a real-world data pipeline, this transformation would be used when there are multiple tenants (e.g., different companies or organizations) sharing the same data platform, and there is a need to analyze sales data in the context of each customer's details. For instance, a sales team might want to analyze the sales performance for each customer across different regions or products.
