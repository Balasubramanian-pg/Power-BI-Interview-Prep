# Append Multiple Tables for Large Datasets

**Description:**
This transformation allows you to combine multiple tables into a single table, which is particularly useful when dealing with large datasets that are split across multiple sources. By appending these tables, you can create a unified view of your data, making it easier to analyze and process. This is especially helpful in data integration and ETL (Extract, Transform, Load) processes.

### Typical Scenario
In a real-world data pipeline, appending multiple tables is often necessary when you have a large dataset that is split across multiple files or databases. For instance, you might have sales data for different regions stored in separate tables. By appending these tables, you can create a single, comprehensive sales dataset that includes data from all regions.
