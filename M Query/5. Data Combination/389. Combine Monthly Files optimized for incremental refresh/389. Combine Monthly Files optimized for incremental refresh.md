# Combine Monthly Files Optimized for Incremental Refresh

**Description:**
This transformation combines monthly files in a Power Query pipeline, optimizing it for incremental refresh. It allows for efficient data updates by only loading new or changed data.

### Typical Scenario
In a real-world data pipeline, this transformation is used when a business needs to aggregate data from multiple monthly files, such as sales reports or financial statements, and wants to optimize the refresh process for incremental updates.
