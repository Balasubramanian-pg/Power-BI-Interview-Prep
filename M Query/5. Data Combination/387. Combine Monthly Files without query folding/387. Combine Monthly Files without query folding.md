# Combine Monthly Files without Query Folding

**Description:**
This use case demonstrates how Power Query (M) can be used to combine monthly files without relying on query folding. It is useful for data pipelines that require the aggregation of data from multiple files, which are typically updated on a monthly basis. By combining these files, users can perform comprehensive analyses and reporting.

### Typical Scenario
In a real-world data pipeline, this transformation is often used when a company receives monthly sales data in separate files. For instance, sales data for January, February, and March are in different Excel files. To analyze the sales trend over these months, it is necessary to combine these files into a single dataset for further analysis and reporting.
