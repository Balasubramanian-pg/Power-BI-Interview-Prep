# Left Join Datasets with Query Folding Considerations

**Description:**
This transformation achieves a left join between two datasets while considering query folding for optimal performance. It allows for efficient merging of data from two sources, preserving all records from the left dataset and matching records from the right dataset.

### Typical Scenario
In a real data pipeline, this transformation is used when integrating customer data from a CRM system with order data from an e-commerce platform. For instance, you want to fetch all customers and their corresponding orders, if any, to analyze customer purchase behavior.
