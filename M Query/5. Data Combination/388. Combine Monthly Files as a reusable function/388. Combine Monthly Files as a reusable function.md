# Combine Monthly Files as a Reusable Function

**Description:**
This transformation combines multiple monthly files into a single dataset using Power Query (M). It enables users to merge data from various files, typically generated on a monthly basis, into one cohesive dataset for easier analysis and reporting.

### Typical Scenario
In a real-world data pipeline, this transformation is useful when a company receives monthly sales data in separate files. By using this reusable function, the company can automate the process of combining these files into a single dataset, making it easier to analyze sales trends over time.

### Core M Functions Used
`Table.Combine`, `Folder.Files`, `Function.Invoke`

### Code Source Execution
```powerquery
let
    Source = Folder.Files("C:\Path\To\MonthlyFiles"),
    CombinedData = Table.Combine(
        List.Transform(Source, each Table.SelectRows(_, {"Column1", "Column2"}))),
    RenamedColumns = Table.RenameColumns(CombinedData, {{ "Column1", "Date" }, { "Column2", "Sales" }})
    in
    RenamedColumns
```

### Example Execution Logic Step-By-Step
The transformation works as follows:
1. **Folder.Files**: This function is used to get a list of files from a specified folder.
2. **List.Transform**: This function transforms each file into a table, selecting only the required columns.
3. **Table.Combine**: This function combines all the individual tables into a single table.
4. **Table.RenameColumns**: This function renames the columns to more meaningful names.
The result is a single table containing data from all monthly files, ready for analysis.
