# Append Multiple Tables as a Reusable Function in Power Query

**Description:**
This transformation allows users to combine multiple tables into a single table, making it easier to analyze and process data from different sources. It is particularly useful when dealing with data from various departments or systems that need to be aggregated for reporting or analysis. By creating a reusable function, users can efficiently append multiple tables with similar structures.

### Typical Scenario
In a real-world data pipeline, this transformation can be used when a company has multiple sales channels (e.g., online, in-store, and wholesale) and wants to combine their sales data into a single dataset for analysis. Each channel's data is stored in a separate table, and by appending these tables, the company can get a comprehensive view of its sales performance across all channels.

```powerquery
let

    Table1 = #"Table 1",
    Table2 = #"Table 2",

    AppendedTable
```

4. Define the list of tables to append.
5. Call the `AppendTables` function with the list of tables.
