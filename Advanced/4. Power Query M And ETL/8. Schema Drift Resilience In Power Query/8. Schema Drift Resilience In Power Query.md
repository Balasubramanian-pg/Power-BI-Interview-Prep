# Schema Drift Resilience in Power Query

## Overview

Schema drift refers to unexpected changes in data structure—new columns appear, existing columns disappear, data types change, or column order shifts. Building resilience against schema drift ensures your Power Query solutions continue working when source systems evolve without requiring manual intervention.

```
Stable Schema → Source Changes → Schema Drift Detection → Adaptive Transformation → Resilient Output
     ↓                 ↓                    ↓                          ↓
  Expected        New columns           Identify                   Handle
  structure       Missing fields         changes                    gracefully
```

## Understanding Schema Drift

### Types of Schema Drift

| Drift Type | Description | Example |
|------------|-------------|---------|
| **New Columns** | Additional fields added | New `CustomerSegment` column |
| **Missing Columns** | Fields removed | `MiddleName` column dropped |
| **Type Changes** | Data format changed | `Price` from number to text |
| **Column Renames** | Field names modified | `FirstName` → `GivenName` |
| **Structural Changes** | Nested structure modified | JSON schema changes |
| **Order Changes** | Column sequence altered | Columns reorganized |

### Why Traditional Approaches Fail

```m
// VULNERABLE: Hard-coded columns
let
    Source = Excel.Workbook(File.Contents("data.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    
    // Assumes these columns always exist and are named exactly this
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Brittle column references
    #"Selected Columns" = Table.SelectColumns(
        #"Promoted Headers", 
        {"ID", "Name", "Price", "Date"}  // Will break if any missing
    ),
    
    // Assumes specific types
    #"Changed Type" = Table.TransformColumnTypes(
        #"Selected Columns",
        {
            {"ID", Int64.Type},
            {"Name", type text},
            {"Price", Currency.Type},
            {"Date", type date}
        }
    )
in
    #"Changed Type"
```

## Pattern 1: Dynamic Column Detection

### Safe Column Selection

```m
let
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data, [PromoteAllScalars=true]),
    
    // Get actual columns from the data
    AvailableColumns = Table.ColumnNames(#"Promoted Headers"),
    
    // Define expected columns (what we want)
    ExpectedColumns = {"OrderID", "CustomerName", "OrderDate", "Amount", "ProductID"},
    
    // Find which expected columns actually exist
    ExistingColumns = List.Intersect({ExpectedColumns, AvailableColumns}),
    
    // Select only existing columns
    SafeSelect = Table.SelectColumns(#"Promoted Headers", ExistingColumns, MissingField.Ignore),
    
    // Add placeholder for missing expected columns
    AddMissingColumns = List.Accumulate(
        List.RemoveItems(ExpectedColumns, ExistingColumns),
        SafeSelect,
        (state, column) => Table.AddColumn(state, column, each null, type any)
    )
in
    AddMissingColumns
```

### Using MissingField.Ignore

```m
let
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Select columns with MissingField.Ignore
    SafeColumns = Table.SelectColumns(
        #"Promoted Headers",
        {"OrderID", "CustomerName", "Email", "Phone", "Amount"},
        MissingField.Ignore  // Missing columns are simply omitted
    )
in
    SafeColumns
```

## Pattern 2: Dynamic Type Handling

### Adaptive Type Detection

```m
let
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Detect types from data
    DetectColumnTypes = (table as table) as list =>
        let
            columnNames = Table.ColumnNames(table),
            
            // Function to infer type from first 100 non-null values
            inferType = (columnName as text) as record =>
                let
                    values = List.RemoveNulls(Table.Column(table, columnName)),
                    sample = List.FirstN(values, 100),
                    
                    // Check type patterns
                    allNumbers = List.AllTrue(List.Transform(sample, each Value.Is(_, type number))),
                    allDates = List.AllTrue(List.Transform(sample, each Value.Is(_, type date))),
                    allText = List.AllTrue(List.Transform(sample, each Value.Is(_, type text))),
                    
                    typeToUse = 
                        if allNumbers then Int64.Type
                        else if allDates then type date
                        else type text
                in
                    [Name = columnName, Type = typeToUse],
            
            types = List.Transform(columnNames, inferType)
        in
            types,
    
    detectedTypes = DetectColumnTypes(#"Promoted Headers"),
    
    // Apply detected types
    typedTable = Table.TransformColumnTypes(
        #"Promoted Headers",
        List.Transform(detectedTypes, each {[Name], [Type]})
    )
in
    typedTable
```

### Safe Type Conversion with Error Handling

```m
let
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Safe conversion function
    SafeConvert = (value as any, targetType as type) as any =>
        try
            if targetType = Int64.Type then
                try Number.From(value) otherwise null
            else if targetType = type date then
                try Date.From(value) otherwise null
            else if targetType = type number then
                try Number.From(value) otherwise null
            else
                value
        otherwise
            null,
    
    // Apply safe conversion to Amount column if it exists
    WithSafeAmount = 
        if List.Contains(Table.ColumnNames(#"Promoted Headers"), "Amount") then
            Table.TransformColumns(
                #"Promoted Headers",
                {{"Amount", each SafeConvert(_, type number), type number}}
            )
        else
            #"Promoted Headers"
in
    WithSafeAmount
```

## Pattern 3: Resilient Column Mapping

### Name-Independent Column Handling

```m
let
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Define column patterns (case-insensitive, partial matching)
    FindColumnByPattern = (table as table, patterns as list) as text =>
        let
            allColumns = Table.ColumnNames(table),
            
            // Try exact match first
            exactMatch = List.Select(
                allColumns,
                each List.AnyTrue(
                    List.Transform(patterns, (pattern) => 
                        Text.Lower(_) = Text.Lower(pattern)
                    )
                )
            ),
            
            // Then try contains match
            containsMatch = if List.IsEmpty(exactMatch) then
                List.Select(
                    allColumns,
                    each List.AnyTrue(
                        List.Transform(patterns, (pattern) =>
                            Text.Contains(Text.Lower(_), Text.Lower(pattern))
                        )
                    )
                )
            else
                exactMatch,
            
            result = if List.IsEmpty(containsMatch) then null else containsMatch{0}
        in
            result,
    
    // Find key columns
    idColumn = FindColumnByPattern(#"Promoted Headers", {"id", "orderid", "order_id", "identifier"}),
    dateColumn = FindColumnByPattern(#"Promoted Headers", {"date", "orderdate", "transactiondate"}),
    amountColumn = FindColumnByPattern(#"Promoted Headers", {"amount", "sales", "value", "price"}),
    
    // Build safe column mapping
    columnsToKeep = List.RemoveNulls({idColumn, dateColumn, amountColumn}),
    
    // Select found columns
    SelectedData = if List.IsEmpty(columnsToKeep) then
        #"Promoted Headers"  // Fallback
    else
        Table.SelectColumns(#"Promoted Headers", columnsToKeep, MissingField.Ignore),
    
    // Rename to standard names
    RenameSteps = [
        (state, mapping) => 
            if mapping[Found] <> null then
                Table.RenameColumns(state, {mapping[Found], mapping[Standard]})
            else
                state
    ],
    
    Final = List.Accumulate(
        {
            [Found = idColumn, Standard = "OrderID"],
            [Found = dateColumn, Standard = "OrderDate"],
            [Found = amountColumn, Standard = "Amount"]
        },
        SelectedData,
        (state, mapping) => 
            if mapping[Found] <> null then
                Table.RenameColumns(state, {{mapping[Found], mapping[Standard]}})
            else
                state
    )
in
    Final
```

## Pattern 4: Schema Comparison and Validation

### Detecting Schema Changes

```m
let
    // Function to get schema of current data
    GetSchema = (table as table) as table =>
        let
            columns = Table.ColumnNames(table),
            
            schemaRecords = List.Transform(
                columns,
                each [
                    ColumnName = _,
                    DataType = Type.RecordFields(Value.Type(table))[_][Type]?,
                    SampleValue = try Table.Column(table, _){0} otherwise null,
                    NullCount = List.Count(List.Select(Table.Column(table, _), each _ = null))
                ]
            ),
            
            schemaTable = Table.FromRecords(schemaRecords)
        in
            schemaTable,
    
    // Load previous schema (store in parameters or table)
    PreviousSchema = try #"StoredSchema" otherwise #table(
        {"ColumnName", "DataType", "NullCount"},
        {}
    ),
    
    // Get current schema
    CurrentSchema = GetSchema(CurrentData),
    
    // Compare schemas
    NewColumns = Table.SelectRows(
        CurrentSchema,
        each not List.Contains(PreviousSchema[ColumnName], [ColumnName])
    ),
    
    MissingColumns = Table.SelectRows(
        PreviousSchema,
        each not List.Contains(CurrentSchema[ColumnName], [ColumnName])
    ),
    
    // Log changes
    Changes = [
        New = Table.RowCount(NewColumns),
        Missing = Table.RowCount(MissingColumns),
        Timestamp = DateTime.LocalNow()
    ]
in
    Changes
```

### Auto-Adapting to Schema Changes

```m
let
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Get current schema
    CurrentColumns = Table.ColumnNames(#"Promoted Headers"),
    
    // Load saved schema from previous run (store in a table)
    SavedSchema = try #"SchemaHistory" otherwise #table(
        {"RunDate", "Columns"},
        {{#date(2024,1,1), {"OrderID", "Date", "Amount"}}}
    ),
    
    // Get most recent schema
    LastRun = Table.Max(SavedSchema, "RunDate"),
    LastColumns = LastRun[Columns],
    
    // Detect new columns
    NewColumns = List.RemoveItems(CurrentColumns, LastColumns),
    
    // Handle new columns
    WithNewColumns = if List.IsEmpty(NewColumns) then
        #"Promoted Headers"
    else
        let
            // Add prefix to new columns to identify them
            AddPrefix = List.Accumulate(
                NewColumns,
                #"Promoted Headers",
                (state, col) => 
                    Table.RenameColumns(state, {{col, "New_" & col}})
            )
        in
            AddPrefix,
    
    // Update schema history (conceptual - would need storage)
    // In practice, store in a separate table or parameter
    
    Final = WithNewColumns
in
    Final
```

## Pattern 5: Resilient JSON Parsing

### Handling Dynamic JSON Structures

```m
let
    Source = Json.Document(File.Contents("C:\data\api_response.json")),
    
    // Resilient path traversal
    SafeNavigate = (json as any, path as list) as any =>
        List.Accumulate(
            path,
            json,
            (current, key) =>
                if current = null then null
                else if Value.Is(current, type record) and Record.HasFields(current, {key}) then
                    current[key]
                else if Value.Is(current, type list) and key = "[]" then
                    current  // Return list for array processing
                else
                    null
        ),
    
    // Extract data with fallbacks
    Data = 
        let
            // Try multiple possible paths
            directData = SafeNavigate(Source, {"data"}),
            resultsData = SafeNavigate(Source, {"results"}),
            itemsData = SafeNavigate(Source, {"items"}),
            
            actualData = 
                if directData <> null then directData
                else if resultsData <> null then resultsData
                else if itemsData <> null then itemsData
                else Source  // Fallback to root
        in
            actualData,
    
    // Ensure we have a list
    DataList = 
        if Value.Is(Data, type list) then Data
        else if Value.Is(Data, type record) then {Data}
        else {},
    
    // Convert to table with all possible fields
    RecordsTable = Table.FromRecords(DataList),
    
    // Handle potential missing fields
    AllFields = List.Distinct(
        List.Combine(
            List.Transform(DataList, each Record.FieldNames(_))
        )
    ),
    
    Final = List.Accumulate(
        AllFields,
        RecordsTable,
        (state, field) =>
            if Table.HasColumns(state, {field}) then
                state
            else
                Table.AddColumn(state, field, each null)
    )
in
    Final
```

## Pattern 6: Multiple Source Schema Reconciliation

### Merging Tables with Different Schemas

```m
let
    // Files from different months with potentially different schemas
    Folder = Folder.Files("C:\data\monthly_reports"),
    ExcelFiles = Table.SelectRows(Folder, each [Extension] = ".xlsx"),
    
    // Function to safely read and normalize a file
    ReadAndNormalize = (filePath as text) as table =>
        let
            Source = Excel.Workbook(File.Contents(filePath), null, true),
            Sheet = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
            #"Promoted Headers" = Table.PromoteHeaders(Sheet),
            
            // Get all columns from this file
            columns = Table.ColumnNames(#"Promoted Headers")
        in
            #"Promoted Headers",
    
    // Read all files
    AllData = Table.AddColumn(
        ExcelFiles,
        "Data",
        each ReadAndNormalize([Folder Path] & [Name]),
        type table
    ),
    
    // Get union of all columns across all files
    AllColumns = List.Distinct(
        List.Combine(
            List.Transform(
                AllData[Data],
                each Table.ColumnNames(_)
            )
        )
    ),
    
    // Function to ensure each table has all columns
    EnsureAllColumns = (tbl as table) as table =>
        List.Accumulate(
            AllColumns,
            tbl,
            (state, col) =>
                if Table.HasColumns(state, {col}) then
                    state
                else
                    Table.AddColumn(state, col, each null)
        ),
    
    // Apply to all tables
    NormalizedData = Table.TransformColumns(
        AllData,
        {{"Data", EnsureAllColumns, type table}}
    ),
    
    // Combine all
    Combined = Table.Combine(NormalizedData[Data])
in
    Combined
```

## Pattern 7: Schema Drift Logging and Alerting

### Monitoring Schema Changes

```m
let
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Get current schema
    CurrentColumns = Table.ColumnNames(#"Promoted Headers"),
    CurrentTypes = List.Transform(
        CurrentColumns,
        each [
            Column = _,
            Type = Type.RecordFields(Value.Type(#"Promoted Headers"))[_][Type]?
        ]
    ),
    
    // Load expected schema (from configuration table)
    ExpectedSchema = #table(
        {"Column", "ExpectedType", "Required"},
        {
            {"OrderID", Int64.Type, true},
            {"Date", type date, true},
            {"Amount", type number, true},
            {"Customer", type text, false},
            {"Region", type text, false}
        }
    ),
    
    // Check for issues
    MissingRequired = Table.SelectRows(
        ExpectedSchema,
        each [Required] = true and 
             not List.Contains(CurrentColumns, [Column])
    ),
    
    NewUnexpected = List.RemoveItems(
        CurrentColumns,
        ExpectedSchema[Column]
    ),
    
    TypeMismatches = List.Select(
        CurrentTypes,
        each 
            let
                expected = Table.SelectRows(ExpectedSchema, (r) => r[Column] = _[Column])[ExpectedType]{0}?
            in
                expected <> null and _[Type] <> expected
    ),
    
    // Log issues
    Issues = [
        Timestamp = DateTime.LocalNow(),
        MissingRequired = Table.RowCount(MissingRequired),
        NewColumns = List.Count(NewUnexpected),
        TypeMismatches = List.Count(TypeMismatches),
        Details = [
            Missing = MissingRequired[Column],
            New = NewUnexpected,
            TypeIssues = TypeMismatches
        ]
    ],
    
    // Raise alert if critical issues
    RaiseAlert = if Table.RowCount(MissingRequired) > 0 then
        error "Critical schema drift detected: missing required columns"
    else
        null,
    
    // Proceed with data loading
    Final = #"Promoted Headers"
in
    Final
```

## Pattern 8: Configuration-Driven Schema Handling

### Using External Configuration

```m
let
    // Load configuration from JSON or table
    Config = Json.Document(File.Contents("C:\config\schema_config.json")),
    
    // Expected structure
    ExpectedColumns = Config[ExpectedColumns],
    ColumnMappings = Config[ColumnMappings],
    DefaultTypes = Config[DefaultTypes],
    
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Apply column mappings (handle renames)
    Renamed = List.Accumulate(
        Record.FieldNames(ColumnMappings),
        #"Promoted Headers",
        (state, sourceCol) =>
            if Table.HasColumns(state, {sourceCol}) then
                Table.RenameColumns(
                    state,
                    {{sourceCol, ColumnMappings[sourceCol]}}
                )
            else
                state
    ),
    
    // Ensure expected columns exist
    WithAllColumns = List.Accumulate(
        ExpectedColumns,
        Renamed,
        (state, col) =>
            if Table.HasColumns(state, {col}) then
                state
            else
                Table.AddColumn(state, col, each null)
    ),
    
    // Apply types from config
    ApplyTypes = List.Accumulate(
        ExpectedColumns,
        WithAllColumns,
        (state, col) =>
            let
                targetType = Record.FieldOrDefault(DefaultTypes, col, type any)
            in
                if targetType <> type any then
                    Table.TransformColumnTypes(state, {{col, targetType}})
                else
                    state
    )
in
    ApplyTypes
```

## Performance Considerations

### Balancing Resilience and Performance

```m
// Check schema infrequently for performance
let
    Source = Excel.Workbook(File.Contents("C:\data\sales.xlsx"), null, true),
    Data = Source{[Item="Sheet1",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(Data),
    
    // Only check schema periodically (e.g., first run of day)
    LastSchemaCheck = try #"LastSchemaCheck" otherwise #date(1900,1,1),
    DaysSinceCheck = Duration.Days(DateTime.Date(DateTime.LocalNow()) - LastSchemaCheck),
    
    // Perform deep schema check only if needed
    SchemaValid = if DaysSinceCheck > 7 then
        let
            // Full schema validation
            CurrentColumns = Table.ColumnNames(#"Promoted Headers"),
            ExpectedColumns = {"OrderID", "Date", "Amount", "Customer"},
            
            // Check if all expected columns exist
            allExist = List.AllTrue(
                List.Transform(ExpectedColumns, each List.Contains(CurrentColumns, _))
            ),
            
            // Update last check timestamp
            _ = #"Update LastSchemaCheck"(DateTime.LocalNow())
        in
            allExist
    else
        true,  // Assume valid
    
    // Proceed with loading
    Result = if SchemaValid then
        #"Promoted Headers"
    else
        error "Schema validation failed"
in
    Result
```

## Best Practices Summary

### Do's
✓ **Use dynamic column detection** - Never hardcode column lists
✓ **Implement fallbacks** - Provide defaults for missing columns
✓ **Log schema changes** - Track drift over time
✓ **Use pattern matching** - Find columns by name patterns
✓ **Validate critical columns** - Ensure required data exists
✓ **Test with varied schemas** - Verify with different column sets
✓ **Document assumptions** - Note which columns are expected

### Don'ts
✗ **Don't assume column order** - Order can change anytime
✗ **Don't hardcode column names** - Use configuration or detection
✗ **Don't ignore type errors** - Handle conversion failures gracefully
✗ **Don't fail on optional columns** - Use MissingField.Ignore
✗ **Don't skip validation** - Always verify critical structure

## Summary

**Key Points for Schema Drift Resilience:**

| Strategy | Implementation |
|----------|----------------|
| **Dynamic Detection** | `Table.ColumnNames`, `List.Intersect` |
| **Safe Selection** | `MissingField.Ignore`, conditional logic |
| **Pattern Matching** | Case-insensitive contains searches |
| **Type Resilience** | Safe conversion with fallbacks |
| **Configuration** | External schema definitions |
| **Monitoring** | Schema comparison logging |
| **Graceful Degradation** | Add missing columns with nulls |

**The Golden Rule:**
> Never assume the schema. Always detect, validate, and adapt. Build queries that expect change and handle it gracefully. The only constant in data is change itself.
