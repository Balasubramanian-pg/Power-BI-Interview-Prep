# M Function Recursion for Hierarchies in Power Query

## Overview

Recursion in Power Query M allows functions to call themselves, making it possible to traverse hierarchical structures of unknown depth. This is essential for processing organizational charts, bill of materials, folder structures, and any parent-child relationships where the depth isn't predetermined.

```
Root Node → Children → Grandchildren → ... → Leaf Nodes
   ↓           ↓            ↓                  ↓
  Level 1    Level 2      Level 3          Final Level
              ↓            ↓                  ↓
            Recursive calls continue until termination condition met
```

## Understanding Recursion in M

### Basic Recursive Structure

```m
// Template for recursive function
let
    RecursiveFunction = (parameter as type) as returnType =>
        if TerminationCondition(parameter) then
            BaseCase
        else
            // Process current node
            let
                Result = ProcessCurrentNode(parameter),
                // Call self with next parameter
                NextLevel = RecursiveFunction(NextParameter)
            in
                CombineResults(Result, NextLevel)
in
    RecursiveFunction
```

### Key Components
| Component | Purpose | Example |
|-----------|---------|---------|
| **Termination Condition** | Stops infinite recursion | `if node = null` |
| **Base Case** | Result when recursion stops | `{}` or `null` |
| **Recursive Call** | Function calling itself | `@FunctionName()` |
| **Parameter Update** | Progress to termination | `child` or `parent` |

## Pattern 1: Folder Hierarchy Traversal

### Get All Files in Nested Folders

```m
let
    // Recursive function to get all files from nested folders
    fn_GetAllFiles = (FolderContent as table) as table =>
        let
            // Get files in current folder
            Files = Table.SelectRows(FolderContent, each [Kind] = "File"),
            
            // Get subfolders
            SubFolders = Table.SelectRows(FolderContent, each [Kind] = "Folder"),
            
            // Recursively process each subfolder
            SubFolderFiles = 
                if Table.IsEmpty(SubFolders) then
                    #table({"Name", "Content"}, {})
                else
                    let
                        // For each subfolder, call this function recursively
                        ProcessSubFolder = (SubFolderRow as record) as table =>
                            let
                                SubFolderContent = SubFolderRow[Content](),
                                NestedFiles = @fn_GetAllFiles(SubFolderContent)
                            in
                                NestedFiles,
                        
                        // Apply to all subfolders
                        AllNestedFiles = Table.Combine(
                            List.Transform(
                                Table.ToRecords(SubFolders),
                                ProcessSubFolder
                            )
                        )
                    in
                        AllNestedFiles,
            
            // Combine current files with subfolder files
            AllFiles = Table.Combine({Files, SubFolderFiles})
        in
            AllFiles,
    
    // Usage
    Source = SharePoint.Contents("https://tenant.sharepoint.com/sites/Sales", [ApiVersion=15]),
    Docs = Source{[Name="Shared Documents"]}[Content],
    AllFiles = fn_GetAllFiles(Docs)
in
    AllFiles
```

### Get Folder Path Hierarchy

```m
let
    // Function to get full path of a folder
    fn_GetFolderPath = (FolderRow as record) as text =>
        let
            // Get folder name
            FolderName = FolderRow[Name],
            
            // Get parent folder (if any)
            Parent = try FolderRow[Parent]() otherwise null,
            
            // Recursively get parent path
            ParentPath = 
                if Parent = null then
                    ""
                else
                    let
                        ParentRecord = 
                            if Value.Is(Parent, type function) then
                                Parent()
                            else
                                Parent,
                        ParentPath = @fn_GetFolderPath(ParentRecord)
                    in
                        if ParentPath = "" then "" else ParentPath & "/",
            
            // Combine
            FullPath = ParentPath & FolderName
        in
            FullPath,
    
    // Usage
    Source = SharePoint.Contents("https://tenant.sharepoint.com/sites/Sales"),
    Docs = Source{[Name="Shared Documents"]}[Content],
    
    // Add folder paths
    WithPaths = Table.AddColumn(
        Table.SelectRows(Docs, each [Kind] = "Folder"),
        "FullPath",
        each fn_GetFolderPath(_),
        type text
    )
in
    WithPaths
```

## Pattern 2: Organizational Chart Traversal

### Build Employee Hierarchy

```m
let
    // Sample employee data
    Employees = #table(
        {"EmployeeID", "Name", "ManagerID"},
        {
            {1, "CEO", null},
            {2, "VP Sales", 1},
            {3, "VP Engineering", 1},
            {4, "Sales Manager", 2},
            {5, "Engineer", 3},
            {6, "Junior Engineer", 5},
            {7, "Intern", 6}
        }
    ),
    
    // Recursive function to get all subordinates
    fn_GetSubordinates = (ManagerID as number) as table =>
        let
            // Direct reports
            DirectReports = Table.SelectRows(
                Employees, 
                each [ManagerID] = ManagerID
            ),
            
            // Recursively get subordinates of each direct report
            Subordinates = 
                if Table.IsEmpty(DirectReports) then
                    #table({"EmployeeID", "Name", "ManagerID", "Level"}, {})
                else
                    let
                        // For each direct report, get their subordinates
                        GetNested = (EmpRow as record) as table =>
                            let
                                EmpID = EmpRow[EmployeeID],
                                EmpName = EmpRow[Name],
                                Nested = @fn_GetSubordinates(EmpID),
                                WithLevel = Table.AddColumn(
                                    Nested,
                                    "Level",
                                    each [Level] + 1,
                                    Int64.Type
                                )
                            in
                                WithLevel,
                        
                        // Apply to all direct reports
                        AllNested = Table.Combine(
                            List.Transform(
                                Table.ToRecords(DirectReports),
                                GetNested
                            )
                        )
                    in
                        AllNested,
            
            // Combine direct reports with nested
            DirectWithLevel = Table.AddColumn(
                DirectReports,
                "Level",
                each 1,
                Int64.Type
            ),
            
            AllReports = Table.Combine({DirectWithLevel, Subordinates})
        in
            AllReports,
    
    // Get all subordinates of CEO
    OrgChart = fn_GetSubordinates(1)
in
    OrgChart
```

### Get Management Chain

```m
let
    Employees = #table(
        {"EmployeeID", "Name", "ManagerID"},
        {
            {1, "CEO", null},
            {2, "VP Sales", 1},
            {3, "VP Engineering", 1},
            {4, "Sales Manager", 2},
            {5, "Engineer", 3},
            {6, "Junior Engineer", 5},
            {7, "Intern", 6}
        }
    ),
    
    // Recursive function to get management chain upward
    fn_GetManagementChain = (EmployeeID as number, Optional IncludeSelf as logical) as table =>
        let
            // Get employee record
            Employee = Table.SelectRows(
                Employees, 
                each [EmployeeID] = EmployeeID
            ),
            
            // Check if employee exists and has manager
            HasManager = not Table.IsEmpty(Employee) and 
                         Employee[ManagerID]{0} <> null,
            
            // Get manager chain recursively
            ManagerChain = 
                if HasManager then
                    @fn_GetManagementChain(Employee[ManagerID]{0}, true)
                else
                    #table({"EmployeeID", "Name", "ManagerID", "Level"}, {}),
            
            // Add current employee
            WithCurrent = 
                if IncludeSelf = true then
                    let
                        WithLevel = Table.AddColumn(
                            Employee,
                            "Level",
                            each 
                                if Table.IsEmpty(ManagerChain) then 0 
                                else Table.Max(ManagerChain, "Level") + 1,
                            Int64.Type
                        )
                    in
                        Table.Combine({ManagerChain, WithLevel})
                else
                    ManagerChain
        in
            WithCurrent,
    
    // Get management chain for intern
    InternChain = fn_GetManagementChain(7, true)
    // Returns: Intern (Level 0), Junior Engineer (Level 1), Engineer (Level 2), 
    //          VP Engineering (Level 3), CEO (Level 4)
in
    InternChain
```

## Pattern 3: Bill of Materials (BOM) Explosion

```m
let
    // Sample BOM data
    BOM = #table(
        {"ParentPart", "ChildPart", "Quantity"},
        {
            {"Product", "Assembly1", 1},
            {"Product", "Assembly2", 2},
            {"Assembly1", "ComponentA", 3},
            {"Assembly1", "ComponentB", 2},
            {"Assembly2", "ComponentC", 1},
            {"Assembly2", "ComponentD", 4},
            {"ComponentD", "RawMaterialX", 5}
        }
    ),
    
    // Recursive function to explode BOM with cumulative quantities
    fn_ExplodeBOM = (ParentPart as text, Optional Multiplier as number) as table =>
        let
            // Set default multiplier
            ActualMultiplier = if Multiplier = null then 1 else Multiplier,
            
            // Get direct children
            Children = Table.SelectRows(
                BOM,
                each [ParentPart] = ParentPart
            ),
            
            // Base case: no children
            NoChildren = Table.IsEmpty(Children),
            
            Result = 
                if NoChildren then
                    // Return leaf node
                    #table(
                        {"Part", "OriginalParent", "CumulativeQuantity", "Level"},
                        {{ParentPart, ParentPart, ActualMultiplier, 0}}
                    )
                else
                    let
                        // For each child, get its children recursively
                        ProcessChild = (ChildRow as record) as table =>
                            let
                                ChildPart = ChildRow[ChildPart],
                                ChildQty = ChildRow[Quantity],
                                CumulativeQty = ActualMultiplier * ChildQty,
                                ChildResult = @fn_ExplodeBOM(ChildPart, CumulativeQty),
                                WithLevel = Table.AddColumn(
                                    ChildResult,
                                    "Level",
                                    each [Level] + 1,
                                    Int64.Type
                                )
                            in
                                WithLevel,
                        
                        // Combine all child results
                        AllComponents = Table.Combine(
                            List.Transform(
                                Table.ToRecords(Children),
                                ProcessChild
                            )
                        ),
                        
                        // Add current level
                        CurrentLevel = Table.AddColumn(
                            #table(
                                {"Part", "OriginalParent", "CumulativeQuantity", "Level"},
                                {{ParentPart, ParentPart, ActualMultiplier, 0}}
                            ),
                            "Level",
                            each 0,
                            Int64.Type
                        )
                    in
                        Table.Combine({CurrentLevel, AllComponents})
        in
            Result,
    
    // Explode entire product
    ProductBOM = fn_ExplodeBOM("Product")
in
    ProductBOM
```

## Pattern 4: Directory Tree Structure

```m
let
    // Create directory tree representation
    fn_BuildDirectoryTree = (Path as text) as table =>
        let
            // Get folder contents
            Folder = Folder.Contents(Path),
            
            // Separate files and folders
            Files = Table.SelectRows(Folder, each [Attributes]?[Kind]? = "File"),
            Folders = Table.SelectRows(Folder, each [Attributes]?[Kind]? = "Folder"),
            
            // Add file records
            FileRecords = Table.AddColumn(
                Files,
                "Type",
                each "File",
                type text
            ),
            
            // Recursively process subfolders
            FolderRecords = 
                if Table.IsEmpty(Folders) then
                    #table({"Name", "FullPath", "Type", "Children"}, {})
                else
                    let
                        ProcessFolder = (FolderRow as record) as record =>
                            let
                                FolderName = FolderRow[Name],
                                FolderPath = FolderRow[Folder Path] & FolderName,
                                Children = @fn_BuildDirectoryTree(FolderPath)
                            in
                                [
                                    Name = FolderName,
                                    FullPath = FolderPath,
                                    Type = "Folder",
                                    Children = Children
                                ],
                        
                        FolderList = List.Transform(
                            Table.ToRecords(Folders),
                            ProcessFolder
                        ),
                        
                        FolderTable = Table.FromRecords(FolderList)
                    in
                        FolderTable,
            
            // Combine files and folders
            AllItems = Table.Combine({
                FileRecords,
                Table.SelectColumns(FolderRecords, {"Name", "FullPath", "Type"})
            })
        in
            AllItems,
    
    // Usage
    DirectoryTree = fn_BuildDirectoryTree("C:\Data\Projects")
in
    DirectoryTree
```

## Pattern 5: JSON/XML Hierarchy Flattening

```m
let
    // Recursive function to flatten nested JSON
    fn_FlattenJSON = (JsonNode as any, Optional ParentPath as text) as table =>
        let
            CurrentPath = if ParentPath = null then "" else ParentPath & ".",
            
            Result = 
                if Value.Is(JsonNode, type record) then
                    // Handle record
                    let
                        FieldNames = Record.FieldNames(JsonNode),
                        
                        ProcessField = (FieldName as text) as table =>
                            let
                                FieldValue = Record.Field(JsonNode, FieldName),
                                FieldResult = @fn_FlattenJSON(FieldValue, CurrentPath & FieldName)
                            in
                                FieldResult,
                        
                        AllFields = Table.Combine(
                            List.Transform(FieldNames, ProcessField)
                        )
                    in
                        AllFields
                
                else if Value.Is(JsonNode, type list) then
                    // Handle list
                    let
                        ListIndices = {0 .. List.Count(JsonNode) - 1},
                        
                        ProcessIndex = (Index as number) as table =>
                            let
                                ItemValue = JsonNode{Index},
                                ItemResult = @fn_FlattenJSON(ItemValue, CurrentPath & "[" & Text.From(Index) & "]")
                            in
                                ItemResult,
                        
                        AllItems = Table.Combine(
                            List.Transform(ListIndices, ProcessIndex)
                        )
                    in
                        AllItems
                
                else
                    // Base case: primitive value
                    #table(
                        {"Path", "Value"},
                        {{CurrentPath, JsonNode}}
                    )
        in
            Result,
    
    // Sample JSON
    SampleJSON = [
        Name = "Product",
        Details = [
            Price = 100,
            InStock = true
        ],
        Tags = {"Electronics", "Sale"},
        Components = {
            [Name = "Part1", Qty = 2],
            [Name = "Part2", Qty = 1]
        }
    ],
    
    // Flatten
    Flattened = fn_FlattenJSON(SampleJSON)
in
    Flattened
```

## Performance Considerations

### Tail Recursion Optimization

M doesn't optimize tail recursion, so deep hierarchies can hit recursion limits :

```m
// Problematic for deep hierarchies
fn_DeepRecursion = (depth) =>
    if depth <= 0 then depth
    else 1 + @fn_DeepRecursion(depth - 1)

// Alternative: iterative approach for very deep structures
fn_IterativeAlternative = (depth) =>
    let
        GenerateList = List.Generate(
            () => 0,
            each _ < depth,
            each _ + 1,
            each _
        ),
        Result = List.Count(GenerateList)
    in
        Result
```

### Recursion Limit

Power Query has a recursion depth limit (approximately 1000 levels). For deeper hierarchies:

```m
// Check depth before recursion
fn_SafeRecursion = (node, currentDepth, maxDepth) =>
    if currentDepth >= maxDepth then
        error "Max depth exceeded: " & Text.From(maxDepth)
    else if TerminationCondition(node) then
        BaseCase
    else
        let
            // Process node
            Result = ProcessNode(node),
            // Recurse with incremented depth
            Next = @fn_SafeRecursion(NextNode(node), currentDepth + 1, maxDepth)
        in
            Combine(Result, Next)
```

### Memory Management

```m
// Process large hierarchies in chunks
fn_ProcessHierarchyInChunks = (rootNode, chunkSize) =>
    let
        // Get all nodes first (if manageable)
        AllNodes = fn_GetAllNodes(rootNode),
        
        // Process in chunks
        Chunks = List.Split(AllNodes, chunkSize),
        
        ProcessChunk = (chunk) =>
            Table.Combine(
                List.Transform(chunk, each ProcessNode(_))
            ),
        
        Results = List.Transform(Chunks, ProcessChunk),
        Combined = Table.Combine(Results)
    in
        Combined
```

## Common Pitfalls and Solutions

### Pitfall 1: Infinite Recursion

```m
// BAD - No termination condition
fn_Infinite = (x) =>
    let
        Result = @fn_Infinite(x + 1)  // Never stops
    in
        Result

// GOOD - Clear termination
fn_Finite = (x, max) =>
    if x >= max then
        x  // Base case
    else
        @fn_Finite(x + 1, max)  // Progress toward base
```

### Pitfall 2: Circular References

```m
// Detect and handle circular references
fn_WithCycleDetection = (node, visitedList) =>
    let
        NodeID = GetNodeID(node),
        
        // Check if already visited
        AlreadyVisited = List.Contains(visitedList, NodeID),
        
        Result = 
            if AlreadyVisited then
                // Cycle detected - return marker
                #table({"CycleDetected", "Node"}, {{true, NodeID}})
            else if IsLeaf(node) then
                ProcessLeaf(node)
            else
                let
                    NewVisited = visitedList & {NodeID},
                    Children = GetChildren(node),
                    
                    ProcessChild = (child) =>
                        @fn_WithCycleDetection(child, NewVisited),
                    
                    AllChildren = Table.Combine(
                        List.Transform(Children, ProcessChild)
                    )
                in
                    AllChildren
    in
        Result
```

### Pitfall 3: Performance with Large Data

```m
// Optimize by reducing function calls
fn_OptimizedHierarchy = (nodes as table, parentColumn as text, childColumn as text) =>
    let
        // Build lookup table once
        ChildrenLookup = Table.Group(
            nodes,
            {parentColumn},
            {{"Children", each _, type table}}
        ),
        
        // Recursive function using lookup
        fn_BuildTree = (parentID) =>
            let
                // Find children using lookup
                ChildRows = try 
                    Table.SelectRows(
                        ChildrenLookup,
                        each Record.Field(_, parentColumn) = parentID
                    )[Children]{0}
                otherwise
                    null,
                
                Result = 
                    if ChildRows = null then
                        null
                    else
                        let
                            // Process each child recursively
                            ProcessedChildren = Table.TransformColumns(
                                ChildRows,
                                {
                                    childColumn,
                                    each fn_BuildTree(_)
                                }
                            )
                        in
                            ProcessedChildren
            in
                Result
    in
        fn_BuildTree
```

## Real-World Example: Complete Hierarchy Processor

```m
// Comprehensive hierarchy processing function with options
let
    fn_ProcessHierarchy = (
        RootNode as any,
        GetChildren as function,
        ProcessNode as function,
        optional Options as record
    ) as table =>
        let
            // Default options
            DefaultOptions = [
                MaxDepth = 100,
                DetectCycles = true,
                IncludePath = true,
                IncludeLevel = true,
                ErrorHandling = "Continue"  // "Continue" or "Stop"
            ],
            
            ActualOptions = if Options = null then DefaultOptions 
                else Record.Combine({DefaultOptions, Options}),
            
            // Main recursive worker
            fn_Traverse = (node, level, path, visited) as table =>
                let
                    // Depth check
                    DepthCheck = 
                        if level > ActualOptions[MaxDepth] then
                            if ActualOptions[ErrorHandling] = "Stop" then
                                error "Max depth exceeded at path: " & path
                            else
                                // Return error row
                                #table(
                                    {"Node", "Level", "Path", "Error"},
                                    {{node, level, path, "MaxDepthExceeded"}}
                                )
                        else null,
                    
                    // Cycle detection
                    CycleCheck = 
                        if ActualOptions[DetectCycles] then
                            let
                                NodeID = GetNodeID(node),
                                CycleDetected = List.Contains(visited, NodeID)
                            in
                                if CycleDetected then
                                    if ActualOptions[ErrorHandling] = "Stop" then
                                        error "Cycle detected at path: " & path
                                    else
                                        // Return error row
                                        #table(
                                            {"Node", "Level", "Path", "Error"},
                                            {{node, level, path, "CycleDetected"}}
                                        )
                                else null
                        else null,
                    
                    // Check for errors
                    HasError = DepthCheck <> null or CycleCheck <> null,
                    
                    Result = 
                        if HasError then
                            if DepthCheck <> null then DepthCheck else CycleCheck
                        else
                            let
                                // Process current node
                                ProcessedNode = ProcessNode(node, level, path),
                                
                                // Add metadata
                                WithMetadata = 
                                    if ActualOptions[IncludePath] and ActualOptions[IncludeLevel] then
                                        Table.AddColumn(
                                            Table.AddColumn(
                                                ProcessedNode,
                                                "Level",
                                                each level,
                                                Int64.Type
                                            ),
                                            "Path",
                                            each path,
                                            type text
                                        )
                                    else if ActualOptions[IncludeLevel] then
                                        Table.AddColumn(
                                            ProcessedNode,
                                            "Level",
                                            each level,
                                            Int64.Type
                                        )
                                    else if ActualOptions[IncludePath] then
                                        Table.AddColumn(
                                            ProcessedNode,
                                            "Path",
                                            each path,
                                            type text
                                        )
                                    else
                                        ProcessedNode,
                                
                                // Get children
                                Children = GetChildren(node),
                                
                                // Process children recursively
                                ChildResults = 
                                    if Children = null or List.IsEmpty(Children) then
                                        #table({}, {})
                                    else
                                        let
                                            // Update visited list
                                            NewVisited = 
                                                if ActualOptions[DetectCycles] then
                                                    visited & {GetNodeID(node)}
                                                else
                                                    visited,
                                            
                                            // Process each child
                                            ProcessChild = (child, index) =>
                                                let
                                                    ChildPath = 
                                                        if ActualOptions[IncludePath] then
                                                            path & "/" & Text.From(index)
                                                        else
                                                            "",
                                                    ChildResult = @fn_Traverse(
                                                        child, 
                                                        level + 1, 
                                                        ChildPath,
                                                        NewVisited
                                                    )
                                                in
                                                    ChildResult,
                                            
                                            AllChildren = Table.Combine(
                                                List.Transform(
                                                    {0 .. List.Count(Children) - 1},
                                                    each ProcessChild(Children{_}, _)
                                                )
                                            )
                                        in
                                            AllChildren,
                                
                                // Combine current with children
                                Combined = 
                                    if Table.IsEmpty(ChildResults) then
                                        WithMetadata
                                    else
                                        let
                                            Compatible = 
                                                if Table.ColumnCount(WithMetadata) = Table.ColumnCount(ChildResults) then
                                                    true
                                                else
                                                    // Pad columns to match
                                                    ChildResults,
                                            
                                            Result = Table.Combine({WithMetadata, ChildResults})
                                        in
                                            Result
                            in
                                Combined
                in
                    Result,
            
            // Start traversal
            FinalResult = fn_Traverse(RootNode, 0, "root", {})
        in
            FinalResult,
    
    // Helper to get node ID (override for your data type)
    GetNodeID = (node) =>
        let
            ID = 
                if Value.Is(node, type record) and Record.HasFields(node, {"ID"}) then
                    node[ID]
                else if Value.Is(node, type text) then
                    node
                else
                    Text.From(node)
        in
            ID,
    
    document = "Generic hierarchy processor with options"
in
    fn_ProcessHierarchy
```

## Summary

**Key Points for M Function Recursion:**

| Aspect | Recommendation |
|--------|----------------|
| **Termination Condition** | Always include clear base case |
| **Parameter Update** | Ensure recursion progresses toward termination |
| **Depth Limit** | Add explicit depth checking |
| **Cycle Detection** | Track visited nodes for circular references |
| **Performance** | Use lookup tables for repeated access |
| **Error Handling** | Graceful handling of recursion errors |
| **Testing** | Test with small, known hierarchies first |

**The Golden Rule:**
> Recursion is powerful for hierarchies, but always include termination conditions, depth limits, and cycle detection. The function must progress toward base case with each call, and the base case must be reachable.
