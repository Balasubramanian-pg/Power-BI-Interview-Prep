# Parsing Complex Dynamic JSON in Power Query

## Overview

Parsing complex JSON with dynamic structure is one of the most challenging tasks in Power Query. Unlike tabular data with consistent schema, complex JSON can have nested objects, arrays, varying structures, and fields that change position or existence. This requires robust parsing strategies that adapt to the data.

```
JSON Document → Schema Detection → Recursive Parsing → Normalized Tables
     ↓                 ↓                    ↓                 ↓
  Nested          Identify            Flatten           Multiple
  Objects         Patterns            Structures        Tables
```

## Understanding JSON Structure Complexity

### Types of JSON Complexity

| Complexity Type | Description | Example |
|-----------------|-------------|---------|
| **Nested Objects** | Objects within objects | `{"user": {"name": "John", "address": {"city": "NYC"}}}` |
| **Arrays of Objects** | Multiple objects in list | `{"users": [{"id": 1}, {"id": 2}]}` |
| **Dynamic Keys** | Field names vary by record | `{"data_2024": {...}, "data_2025": {...}}` |
| **Mixed Types** | Same field can be different types | `{"value": 123}` vs `{"value": "N/A"}` |
| **Sparse Fields** | Fields appear in some records but not others | Some have "email", some don't |
| **Deep Nesting** | Many levels of hierarchy | Level1.Level2.Level3...LevelN |

## Core Parsing Functions

```m
// Essential functions for JSON parsing
let
    // Parse JSON from text or binary
    JsonFromText = (jsonText as text) => Json.Document(jsonText),
    JsonFromBinary = (binary as binary) => Json.Document(binary),
    
    // Type checking
    IsRecord = (value as any) => Value.Is(value, type record),
    IsList = (value as any) => Value.Is(value, type list),
    IsPrimitive = (value as any) => not (IsRecord(value) or IsList(value)),
    
    // Safe field access
    GetField = (record as record, field as text, optional default as any) =>
        if Record.HasFields(record, field) then record[field] else default
in
    [JsonFromText = JsonFromText, JsonFromBinary = JsonFromBinary]
```

## Pattern 1: Flattening Nested JSON

### Basic Recursive Flattener

```m
let
    // Recursive function to flatten nested JSON
    FlattenJson = (json as any, optional parentPath as text) as table =>
        let
            currentPath = if parentPath = null then "" else parentPath & ".",
            
            result = 
                if Value.Is(json, type record) then
                    // Handle record - flatten each field
                    let
                        fieldNames = Record.FieldNames(json),
                        
                        // Process each field
                        processField = (fieldName as text) as table =>
                            let
                                fieldValue = Record.Field(json, fieldName),
                                fieldResult = @FlattenJson(fieldValue, currentPath & fieldName)
                            in
                                fieldResult,
                        
                        // Combine all fields
                        allFields = Table.Combine(List.Transform(fieldNames, processField))
                    in
                        allFields
                
                else if Value.Is(json, type list) then
                    // Handle list - add index to each element
                    let
                        indices = {0 .. List.Count(json) - 1},
                        
                        processIndex = (idx as number) as table =>
                            let
                                element = json{idx},
                                elementResult = @FlattenJson(element, currentPath & "[" & Text.From(idx) & "]")
                            in
                                elementResult,
                        
                        allElements = Table.Combine(List.Transform(indices, processIndex))
                    in
                        allElements
                
                else
                    // Handle primitive value
                    #table(
                        {"Path", "Value"},
                        {{currentPath, json}}
                    )
        in
            result,
    
    // Usage
    SampleJson = [
        id = 123,
        name = "Product",
        details = [
            price = 99.99,
            inStock = true
        ],
        tags = {"electronics", "sale"}
    ],
    
    Flattened = FlattenJson(SampleJson)
in
    Flattened
```

### Path-Based Table Transformation

```m
let
    // Convert flattened path table to structured columns
    PathToColumns = (flattenedTable as table) as table =>
        let
            // Group by root paths
            AddRootColumn = Table.AddColumn(flattenedTable, "Root",

            )

            Pivoted
```

## Pattern 2: Handling Dynamic Keys

### Schema Detection

```m
let
    // Detect all possible fields in a collection
    DetectSchema = (jsonList as list) as list =>
        let
            // Function to get all field paths from a record
            GetAllPaths = (item as any, optional path as text) as list =>
                let
                    currentPath = if path = null then "" else path & ".",
                    
                    result = 
                        if Value.Is(item, type record) then
                            let
                                fieldNames = Record.FieldNames(item),
                                
                                // Process each field
                                processField = (fieldName as text) =>
                                    let
                                        fieldValue = Record.Field(item, fieldName),
                                        fieldPaths = @GetAllPaths(fieldValue, currentPath & fieldName)
                                    in
                                        fieldPaths,
                                
                                allPaths = List.Combine(List.Transform(fieldNames, processField))
                            in
                                allPaths
                        
                        else if Value.Is(item, type list) then
                            // For lists, use first element as representative
                            if List.IsEmpty(item) then
                                {currentPath & "[]"}
                            else
                                @GetAllPaths(item{0}, currentPath & "[]")
                        
                        else
                            {currentPath}
                in
                    result,
            
            // Get paths from all items
            allPaths = List.Combine(List.Transform(jsonList, each GetAllPaths(_))),
            
            // Get unique paths
            uniquePaths = List.Distinct(allPaths)
        in
            uniquePaths,
    
    // Usage
    SampleData = {
        [id = 1, name = "A", attributes = [color = "red", size = "M"]],
        [id = 2, name = "B", attributes = [color = "blue", weight = 10]],
        [id = 3, name = "C", extra = [note = "test"]]
    },
    
    Schema = DetectSchema(SampleData)
    // Returns: {"id", "name", "attributes.color", "attributes.size", 
    //           "attributes.weight", "extra.note"}
in
    Schema
```

### Dynamic Record Expansion

```m
let
    // Expand records dynamically based on detected schema
    ExpandDynamically = (table as table, recordColumn as text) as table =>
        let
            // Get all records from column
            records = Table.Column(table, recordColumn),
            
            // Detect all possible fields
            allFields = List.Distinct(
                List.Combine(
                    List.Transform(
                        List.Select(records, each _ <> null),
                        each Record.FieldNames(_)
                    )
                )
            ),
            
            // Add each field as a column
            result = List.Accumulate(
                allFields,
                table,
                (state, field) =>
                    if Table.HasColumns(state, {field}) then
                        state
                    else
                        Table.AddColumn(
                            state,
                            field,
                            each if _[recordColumn] = null then null 
                                 else Record.Field(_[recordColumn], field)
                        )
            )
        in
            result,
    
    // Usage
    Source = Table.FromRecords({
        [ID = 1, Data = [Name = "A", Value = 100]],
        [ID = 2, Data = [Name = "B", Category = "X"]],
        [ID = 3, Data = [Name = "C", Value = 200, Active = true]]
    }),
    
    Expanded = ExpandDynamically(Source, "Data")
in
    Expanded
```

## Pattern 3: Handling Arrays of Varying Objects

### Normalizing Array Data

```m
let
    // Normalize array of objects with varying fields
    NormalizeArray = (jsonArray as list) as table =>
        let
            // Convert to table
            sourceTable = Table.FromRecords(jsonArray),
            
            // Get all possible columns
            allColumns = List.Distinct(
                List.Combine(
                    List.Transform(jsonArray, each Record.FieldNames(_))
                )
            ),
            
            // Ensure all columns exist
            withAllColumns = List.Accumulate(
                allColumns,
                sourceTable,
                (state, column) =>
                    if Table.HasColumns(state, {column}) then
                        state
                    else
                        Table.AddColumn(state, column, each null)
            ),
            
            // Reorder columns for consistency
            reordered = Table.SelectColumns(withAllColumns, allColumns)
        in
            reordered,
    
    // Usage
    SampleArray = {
        [id = 1, name = "A", price = 100],
        [id = 2, name = "B", category = "X"],
        [id = 3, name = "C", price = 200, inStock = true]
    },
    
    Normalized = NormalizeArray(SampleArray)
in
    Normalized
```

### Splitting Arrays by Type

```m
let
    // Split array based on record structure
    SplitArrayByStructure = (jsonArray as list) as record =>
        let
            // Group records by their field set
            recordsWithSchema = List.Transform(
                jsonArray,
                each [Record = _, Fields = Record.FieldNames(_)]
            ),
            
            // Group by field set
            grouped = List.Accumulate(
                recordsWithSchema,
                [],
                (state, item) =>
                    let
                        fieldsKey = Text.Combine(List.Sort(item[Fields]), "|"),
                        currentGroup = Record.FieldOrDefault(state, fieldsKey, {}),
                        newGroup = currentGroup & {item[Record]}
                    in
                        Record.AddField(state, fieldsKey, newGroup)
            ),
            
            // Convert each group to table
            result = Record.TransformFields(
                grouped,
                List.Transform(
                    Record.FieldNames(grouped),
                    each {_, (list) => Table.FromRecords(list)}
                )
            )
        in
            result,
    
    // Usage
    MixedArray = {
        [type = "A", id = 1],
        [type = "A", id = 2, value = 100],
        [type = "B", code = "X"],
        [type = "A", id = 3, value = 200]
    },
    
    Split = SplitArrayByStructure(MixedArray)
    // Returns record with tables for each schema type
in
    Split
```

## Pattern 4: Streaming Parser for Large JSON

For large JSON files that don't fit in memory:

```m
let
    // Stream-process large JSON array

            // Extract array content
            arrayContent = Text.Range(fullText, arrayStart + 1, arrayEnd - arrayStart - 1),
            
            // Split into chunks (simplified - assumes objects separated by commas)
            SplitIntoChunks = (text as text, size as number) as list =>
                let
                    // This is simplified - real implementation needs proper JSON parsing
                    positions = List.Generate(
                        () => [Start = 0, Level = 0],
                        each [Start] < Text.Length(text),
                        each 
                            let
                                // Find next object boundary
                                // Complex implementation omitted for brevity
                                nextPos = Text.Length(text)
                            in
                                [Start = nextPos + 1, Level = 0],
                        each Text.Range(text, [Start], 1000)
                    )
                in
                    positions,
            
            chunks = SplitIntoChunks(arrayContent, chunkSize),
```

## Pattern 5: Handling Mixed Types

### Type Coercion and Normalization

```m
let
    // Normalize mixed type fields
    NormalizeField = (table as table, fieldName as text) as table =>
        let
            // Get all values
            values = Table.Column(table, fieldName),
            
            // Determine target type (prefer number if most are numbers)
            typeCounts = List.Accumulate(
                values,
                [Number = 0, Text = 0, Other = 0],
                (state, value) =>
                    if Value.Is(value, type number) then
                        state & [Number = state[Number] + 1]
                    else if Value.Is(value, type text) then
                        state & [Text = state[Text] + 1]
                    else
                        state & [Other = state[Other] + 1]
            ),
            
            targetType = 
                if typeCounts[Number] > typeCounts[Text] then
                    type number
                else
                    type text,
            
            // Transform values
            transformFunc = 
                if targetType = type number then
                    (value) => 
                        if Value.Is(value, type number) then value
                        else if Value.Is(value, type text) then
                            try Number.From(value) otherwise null
                        else null
                else
                    (value) => 
                        if Value.Is(value, type text) then value
                        else if Value.Is(value, type number) then
                            Text.From(value)
                        else null,
            
            // Replace column
            transformed = Table.TransformColumns(
                table,
                {fieldName, transformFunc, targetType}
            )
        in
            transformed,
    
    // Usage
    Source = Table.FromRecords({
        [ID = 1, Value = 100],
        [ID = 2, Value = "200"],
        [ID = 3, Value = true],
        [ID = 4, Value = "N/A"]
    }),
    
    Normalized = NormalizeField(Source, "Value")
in
    Normalized
```

## Pattern 6: Recursive Record Expansion

```m
let
    // Recursively expand all record columns
    ExpandAllRecords = (table as table) as table =>
        let
            // Find record-type columns
            columnNames = Table.ColumnNames(table),
            
            // Check each column for record type
            isRecordColumn = (colName as text) as logical =>
                let
                    sample = try Table.Column(table, colName){0} otherwise null
                in
                    sample <> null and Value.Is(sample, type record),
            
            recordColumns = List.Select(columnNames, each isRecordColumn(_)),
            
            // If no record columns, return table
            result = if List.IsEmpty(recordColumns) then
                table
            else
                let
                    // Expand first record column
                    firstColumn = recordColumns{0},
                    
                    // Get fields from first non-null record
                    records = List.Select(Table.Column(table, firstColumn), each _ <> null),
                    fields = if List.IsEmpty(records) then {} 
                            else Record.FieldNames(records{0}),
                    
                    // Expand column
                    expanded = Table.ExpandRecordColumn(
                        table,
                        firstColumn,
                        fields,
                        List.Transform(fields, each firstColumn & "." & _)
                    ),
                    
                    // Recursively expand remaining
                    recursive = @ExpandAllRecords(expanded)
                in
                    recursive
        in
            result,
    
    // Usage
    Source = Table.FromRecords({
        [ID = 1, Data = [Name = "A", Details = [Price = 100, InStock = true]]],
        [ID = 2, Data = [Name = "B", Details = [Price = 200]]]
    }),
    
    FullyExpanded = ExpandAllRecords(Source)
in
    FullyExpanded
```

## Pattern 7: Handling JSON with Metadata

```m
let
    // Parse JSON that includes metadata and data sections
    ParseWithMetadata = (json as record) as record =>
        let
            // Common patterns
            metadata = 
                if Record.HasFields(json, {"metadata", "data"}) then
                    [Metadata = json[metadata], Data = json[data]]
                else if Record.HasFields(json, {"meta", "result"}) then
                    [Metadata = json[meta], Data = json[result]]
                else if Record.HasFields(json, {"pagination", "items"}) then
                    [Metadata = json[pagination], Data = json[items]]
                else
                    [Metadata = [], Data = json],
            
            // Process data based on type
            processedData = 
                if Value.Is(metadata[Data], type list) then
                    Table.FromRecords(metadata[Data])
                else if Value.Is(metadata[Data], type record) then
                    Table.FromRecords({metadata[Data]})
                else
                    #table({"Value"}, {{metadata[Data]}}),
            
            // Add metadata as columns
            withMetadata = List.Accumulate(
                Record.FieldNames(metadata[Metadata]),
                processedData,
                (state, field) =>
                    Table.AddColumn(
                        state,
                        "meta_" & field,
                        each metadata[Metadata][field]
                    )
            )
        in
            withMetadata,
    
    // Usage
    SampleJson = [
        metadata = [
            totalCount = 100,
            page = 1,
            pageSize = 10
        ],
        data = {
            [id = 1, name = "A"],
            [id = 2, name = "B"]
        }
    ],
    
    Parsed = ParseWithMetadata(SampleJson)
in
    Parsed
```

## Pattern 8: Performance Optimization

### Caching Schema Detection

```m
let
    // Cache schema to avoid repeated detection
    CachedSchemaParser = (jsonList as list) as table =>
        let
            // Detect schema once
            schema = List.Distinct(
                List.Combine(
                    List.Transform(jsonList, each Record.FieldNames(_))
                )
            ),
            
            // Function to normalize a single record using cached schema
            NormalizeRecord = (record as record) as record =>
                let
                    // Start with all fields null
                    baseRecord = Record.FromList(
                        List.Repeat({null}, List.Count(schema)),
                        schema
                    ),
                    
                    // Override with actual values
                    fullRecord = Record.Combine({baseRecord, record})
                in
                    fullRecord,
            
            // Apply to all records
            normalizedRecords = List.Transform(jsonList, NormalizeRecord),
            result = Table.FromRecords(normalizedRecords)
        in
            result,
    
    // Usage
    LargeArray = List.Repeat(
        {[id = 1, name = "A", value = 100], [id = 2, category = "B"]},
        1000
    ),
    
    Parsed = CachedSchemaParser(LargeArray)
in
    Parsed
```

## Real-World Example: Complete JSON Parser

```m
let
    // Complete JSON parsing solution
    ParseComplexJson = (
        jsonInput as any,
        optional options as record
    ) as table =>
        let
            // Default options
            defaultOptions = [
                Flatten = true,
                DetectSchema = true,
                ExpandArrays = true,
                MaxDepth = 10,
                ErrorHandling = "Continue"  // "Continue" or "Stop"
            ],
            
            actualOptions = if options = null then defaultOptions 
                           else Record.Combine({defaultOptions, options}),
            
            // Parse input to JSON
            json = 
                if Value.Is(jsonInput, type text) then
                    try Json.Document(jsonInput) otherwise 
                        error "Invalid JSON text"
                else if Value.Is(jsonInput, type binary) then
                    try Json.Document(jsonInput) otherwise
                        error "Invalid JSON binary"
                else
                    jsonInput,
            
            // Ensure we have a list
            jsonList = 
                if Value.Is(json, type list) then
                    json
                else if Value.Is(json, type record) then
                    {json}
                else
                    error "JSON must be object or array",
            
            // Detect schema if needed
            schema = if actualOptions[DetectSchema] then
                let
                    getAllPaths = (item, path) =>
                        // ... schema detection logic
                        {},
                    
                    allPaths = List.Distinct(
                        List.Combine(
                            List.Transform(jsonList, each getAllPaths(_, ""))
                        )
                    )
                in
                    allPaths
            else
                {},
            
            // Parse based on options
            parsed = 
                if actualOptions[Flatten] then
                    let
                        // Flatten each record
                        flattenRecord = (record) =>
                            let
                                // Flattening logic
                                flattened = []
                            in
                                flattened,
                        
                        flattenedList = List.Transform(jsonList, flattenRecord),
                        result = Table.FromRecords(flattenedList)
                    in
                        result
                else
                    Table.FromRecords(jsonList),
            
            // Handle errors
            final = 
                if actualOptions[ErrorHandling] = "Continue" then
                    try parsed otherwise #table({"Error"}, {{"Parsing failed"}})
                else
                    parsed
        in
            final,
    
    // Usage
    SampleJson = "[
        {""id"": 1, ""name"": ""A"", ""details"": {""price"": 100}},
        {""id"": 2, ""name"": ""B"", ""tags"": [""x"", ""y""]}
    ]",
    
    Result = ParseComplexJson(SampleJson, [Flatten = true, MaxDepth = 5])
in
    Result
```

## Error Handling Strategies

### Graceful Degradation

```m
let
    SafeParse = (jsonText as text) as table =>
        let
            // Try multiple parsing strategies
            result = try
                // Strategy 1: Direct parse
                Json.Document(jsonText)
            otherwise try
                // Strategy 2: Fix common issues
                let
                    // Replace single quotes with double quotes
                    fixed = Text.Replace(jsonText, "'", "\""),
                    parsed = Json.Document(fixed)
                in
                    parsed
            otherwise try
                // Strategy 3: Extract JSON from text
                let
                    start = Text.PositionOf(jsonText, "{"),
                    end = Text.LastPositionOf(jsonText, "}"),
                    extracted = Text.Range(jsonText, start, end - start + 1),
                    parsed = Json.Document(extracted)
                in
                    parsed
            otherwise
                // Fallback: Return error table
                #table({"Error", "OriginalText"}, {{"Parse failed", Text.Start(jsonText, 100)}}),
            
            // Convert to table if needed
            final = 
                if Value.Is(result, type table) then
                    result
                else if Value.Is(result, type list) then
                    Table.FromRecords(result)
                else if Value.Is(result, type record) then
                    Table.FromRecords({result})
                else
                    #table({"Value"}, {{result}})
        in
            final
in
    SafeParse
```

## Performance Considerations

| Factor | Impact | Mitigation |
|--------|--------|------------|
| **JSON Size** | Large files can exceed memory | Stream or chunk processing |
| **Nesting Depth** | Deep recursion can be slow | Limit depth, use iterative approach |
| **Array Size** | Large arrays cause memory pressure | Process in batches |
| **Schema Detection** | Scanning all records is expensive | Sample first, cache schema |
| **Multiple Expansions** | Repeated expansions are costly | Expand once, reuse |

## Best Practices Summary

### Do's
✓ **Detect schema first** - Understand structure before parsing
✓ **Cache detected schema** - Avoid repeated detection
✓ **Handle missing fields** - Use null defaults
✓ **Validate JSON** - Check for common issues
✓ **Limit recursion depth** - Prevent stack overflow
✓ **Test with samples** - Verify with representative data
✓ **Document structure** - Comment on expected JSON format

### Don'ts
✗ **Don't assume consistent structure** - Always handle variations
✗ **Don't expand all at once** - Incremental approach
✗ **Don't ignore nulls** - Handle missing values explicitly
✗ **Don't use deep recursion on large data** - Stack limits
✗ **Don't parse repeatedly** - Buffer parsed results

## Summary

**Key Points for Parsing Complex Dynamic JSON:**

| Challenge | Solution |
|-----------|----------|
| **Nested objects** | Recursive flattening |
| **Dynamic keys** | Schema detection |
| **Varying arrays** | Normalization with all fields |
| **Mixed types** | Type coercion with majority rule |
| **Deep hierarchies** | Path-based flattening |
| **Large files** | Streaming or chunking |
| **Performance** | Cached schema, buffered results |

**The Golden Rule:**
> Complex JSON requires adaptive parsing. Never assume structure—detect it. Never hardcode paths—derive them. And always, always handle missing fields gracefully. The only constant in dynamic JSON is change.
