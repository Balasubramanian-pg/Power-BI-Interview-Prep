# Binary File Processing in Power Query

## Overview

Binary file processing in Power Query involves reading, extracting, and transforming data from files stored in binary format. This includes Excel files, PDFs, images, compressed archives, and custom binary formats. Power Query provides several functions for handling binary data, from simple file reading to complex binary parsing.

```
File Source → Binary Content → Format Detection → Parsing → Structured Data
    ↓               ↓                 ↓              ↓
SharePoint,   Raw bytes        Excel.Workbook,   Tables,
Local Path    (binary type)     Pdf.Tables        Text
```

## Core Binary Functions

| Function | Purpose | Example |
|----------|---------|---------|
| **File.Contents** | Read file from local path | `File.Contents("C:\data.xlsx")` |
| **Binary.Buffer** | Cache binary in memory | `Binary.Buffer(Content)` |
| **Binary.Combine** | Combine multiple binaries | `Binary.Combine({part1, part2})` |
| **Binary.Format** | Parse binary using format spec | Advanced binary parsing |
| **Binary.InferContents** | Detect content type | Identify file type |
| **Binary.ToText** | Convert to text | `Binary.ToText(Content, TextEncoding.Utf8)` |
| **Binary.ToList** | Convert to list of bytes | `Binary.ToList(Content)` |
| **Text.ToBinary** | Convert text to binary | `Text.ToBinary("Hello")` |
| **Binary.Range** | Extract portion of binary | `Binary.Range(Content, 0, 100)` |

## Pattern 1: Reading Binary Files

### From Local File System

```m
let
    // Read entire file as binary
    BinaryContent = File.Contents("C:\Data\Report.xlsx"),
    
    // Process based on file type
    ExcelData = Excel.Workbook(BinaryContent, null, true),
    
    // Or read as text for text files
    TextContent = Binary.ToText(File.Contents("C:\Data\log.txt"), TextEncoding.Utf8)
in
    ExcelData
```

### From SharePoint

```m
let
    Source = SharePoint.Contents(
        "https://tenant.sharepoint.com/sites/Sales",
        [ApiVersion=15]
    ),
    Documents = Source{[Name="Shared Documents"]}[Content],
    
    // Binary content is in the [Content] column
    FileList = Table.SelectRows(Documents, each [Extension] = ".xlsx"),
    
    // Process each binary
    ProcessFile = (binary as binary) as table =>
        Excel.Workbook(binary, null, true),
    
    WithData = Table.AddColumn(
        FileList,
        "Data",
        each ProcessFile([Content]),
        type table
    )
in
    WithData
```

### From Web API

```m
let
    // Download binary file from URL
    BinaryContent = Web.Contents("https://example.com/files/report.pdf"),
    
    // PDF parsing (requires custom parser)
    // Power Query doesn't have native PDF support
    Text = Binary.ToText(BinaryContent, TextEncoding.Ascii)
in
    Text
```

## Pattern 2: Excel Binary Processing

### Reading Specific Sheets

```m
let
    BinaryContent = File.Contents("C:\Data\Sales.xlsx"),
    
    // Read workbook
    Workbook = Excel.Workbook(BinaryContent, null, true),
    
    // Get specific sheet
    Sheet1 = Table.SelectRows(Workbook, each [Name] = "Sheet1")[Data]{0},
    
    // Get named table
    SalesTable = Table.SelectRows(Workbook, each [Name] = "SalesData" and [Kind] = "Table")[Data]{0},
    
    // Promote headers and set types
    Promoted = Table.PromoteHeaders(SalesTable, [PromoteAllScalars=true]),
    Typed = Table.TransformColumnTypes(Promoted, {
        {"Date", type date},
        {"Amount", type number}
    })
in
    Typed
```

### Combining Multiple Excel Files

```m
let
    // Get list of Excel files
    Folder = Folder.Files("C:\Data\Monthly Reports"),
    ExcelFiles = Table.SelectRows(Folder, each [Extension] = ".xlsx"),
    
    // Function to extract data from one file
    ExtractData = (fileBinary as binary) as table =>
        let
            Workbook = Excel.Workbook(fileBinary, null, true),
            Sheet = Table.SelectRows(Workbook, each [Kind] = "Sheet"){0}[Data],
            Promoted = Table.PromoteHeaders(Sheet, [PromoteAllScalars=true])
        in
            Promoted,
    
    // Apply to all files
    WithData = Table.AddColumn(
        ExcelFiles,
        "Data",
        each ExtractData([Content]),
        type table
    ),
    
    // Expand all data
    Expanded = Table.ExpandTableColumn(
        WithData,
        "Data",
        {"Date", "Product", "Amount"},
        {"Date", "Product", "Amount"}
    ),
    
    // Add source file name
    WithSource = Table.AddColumn(Expanded, "SourceFile", each [Name], type text)
in
    WithSource
```

## Pattern 3: CSV and Text File Processing

### Reading Delimited Text

```m
let
    BinaryContent = File.Contents("C:\Data\export.csv"),
    
    // Convert to text and split
    Text = Binary.ToText(BinaryContent, TextEncoding.Utf8),
    
    // Manual parsing (if Csv.Document doesn't work)
    Lines = Lines.FromText(Text),
    Headers = Lines{0},
    DataRows = List.Skip(Lines, 1),
    
    // Parse each row
    ParseRow = (row as text) as list =>
        Text.Split(row, ","),
    
    RowLists = List.Transform(DataRows, each ParseRow(_)),
    
    // Convert to table
    Result = Table.FromRows(RowLists, Text.Split(Headers, ","))
in
    Result
```

### Custom ZIP Entry Extraction

For proper ZIP handling, you need to parse the ZIP structure:

```m
let
    // Read ZIP binary
    ZipBinary = File.Contents("C:\Data\archive.zip"),
    
    // Find End of Central Directory Record
    FindEOCD = (binary as binary) as number =>
        let
            size = Binary.Length(binary),
            searchStart = size - 22,  // Minimum EOCD size
            // Search for PK\5\6 signature (0x06054b50)
            eocdOffset = 
                List.Accumulate(
                    {0..20},
                    -1,
                    (state, offset) =>
                        if state >= 0 then state
                        else
                            let
                                pos = searchStart + offset,
                                bytes = Binary.Range(binary, pos, 4),
                                signature = Binary.ToList(bytes)
                            in
                                if signature = {0x50, 0x4b, 0x05, 0x06} then pos else -1
                )
        in
            eocdOffset,
    
    // Parse EOCD record
    ParseEOCD = (binary as binary, offset as number) as record =>
        let
            eocd = Binary.Range(binary, offset, 22),
            bytes = Binary.ToList(eocd),
            
            // Parse fields (little-endian)
            diskNumber = bytes{4} + bytes{5} * 256,
            centralDirDisk = bytes{6} + bytes{7} * 256,
            entriesOnDisk = bytes{8} + bytes{9} * 256,
            totalEntries = bytes{10} + bytes{11} * 256,
            centralDirSize = bytes{12} + bytes{13} * 256 + bytes{14} * 65536 + bytes{15} * 16777216,
            centralDirOffset = bytes{16} + bytes{17} * 256 + bytes{18} * 65536 + bytes{19} * 16777216,
            commentLength = bytes{20} + bytes{21} * 256
        in
            [
                TotalEntries = totalEntries,
                CentralDirOffset = centralDirOffset,
                CentralDirSize = centralDirSize,
                CommentLength = commentLength
            ],
    
    // Find and parse central directory
    eocdOffset = FindEOCD(ZipBinary),
    eocd = if eocdOffset >= 0 then ParseEOCD(ZipBinary, eocdOffset) else null,
    
    // List files in ZIP (simplified - real implementation would parse central directory entries)
    FileList = if eocd = null then {} else {eocd[TotalEntries]}
in
    FileList
```

## Pattern 5: PDF Processing

Power Query doesn't have native PDF support. For PDFs, you need to:

### Option A: Extract Text (Basic)

```m
let
    PdfBinary = File.Contents("C:\Data\document.pdf"),
    
    // Convert to text (will include binary garbage)
    Text = Binary.ToText(PdfBinary, TextEncoding.Ascii),
    
    // Look for text sections (simplified)
    // PDF text is typically in streams with markers
    FindTextStreams = (pdfText as text) as list =>
        let
            // Find BT/ET markers (Begin Text/End Text)
            // This is highly simplified - real PDF parsing is complex
            lines = Lines.FromText(pdfText),
            textLines = List.Select(lines, each Text.Contains(_, "BT") or Text.Contains(_, "ET"))
        in
            textLines
in
    FindTextStreams(Text)
```

### Option B: Use External Service

```m
let
    // Send PDF to external parsing service
    PdfBinary = File.Contents("C:\Data\document.pdf"),
    
    // Convert to base64 for API transmission
    Base64Content = Binary.ToText(PdfBinary, BinaryEncoding.Base64),
    
    // Call PDF parsing API
    ApiResponse = Web.Contents(
        "https://api.pdfparser.com/v1/extract",
        [
            Headers = [
                Authorization = "Bearer YOUR_API_KEY",
                #"Content-Type" = "application/json"
            ],
            Content = Json.FromValue([
                file = Base64Content,
                extractTables = true
            ])
        ]
    ),
    
    JsonResponse = Json.Document(ApiResponse),
    Tables = JsonResponse[tables]
in
    Tables
```

## Pattern 6: Image File Processing

### Extract Image Metadata

```m
let
    // Read image file
    ImageBinary = File.Contents("C:\Data\photo.jpg"),
    
    // Get file size
    Size = Binary.Length(ImageBinary),
    
    // JPEG files start with FF D8, end with FF D9
    FirstBytes = Binary.ToList(Binary.Range(ImageBinary, 0, 2)),
    IsJPEG = FirstBytes = {0xFF, 0xD8},
    
    // Try to find EXIF data (simplified)
    FindEXIF = (binary as binary) as record =>
        let
            bytes = Binary.ToList(binary),
            // Search for EXIF header "Exif" (0x45786966)
            findExif = 
                List.PositionOf(
                    List.Transform({0..List.Count(bytes)-4}, each bytes{_}),
                    0x45  // 'E'
                ),
            
            result = if findExif >= 0 then
                [HasEXIF = true, Position = findExif]
            else
                [HasEXIF = false]
        in
            result,
    
    EXIFInfo = FindEXIF(ImageBinary)
in
    [Size = Size, IsJPEG = IsJPEG, EXIF = EXIFInfo]
```

### Extract PNG Dimensions

```m
let
    PngBinary = File.Contents("C:\Data\image.png"),
    
    // PNG header: 137 80 78 71 13 10 26 10
    // IHDR chunk starts at position 8, contains width/height at positions 16-23
    GetDimensions = (binary as binary) as record =>
        let
            // Check PNG signature
            header = Binary.ToList(Binary.Range(binary, 0, 8)),
            isPNG = header = {137, 80, 78, 71, 13, 10, 26, 10},
            
            dimensions = if isPNG then
                let
                    // Width is bytes 16-19
                    widthBytes = Binary.ToList(Binary.Range(binary, 16, 4)),
                    width = widthBytes{0} * 16777216 + widthBytes{1} * 65536 + 
                            widthBytes{2} * 256 + widthBytes{3},
                    
                    // Height is bytes 20-23
                    heightBytes = Binary.ToList(Binary.Range(binary, 20, 4)),
                    height = heightBytes{0} * 16777216 + heightBytes{1} * 65536 + 
                             heightBytes{2} * 256 + heightBytes{3}
                in
                    [Width = width, Height = height]
            else
                [Error = "Not a valid PNG"]
        in
            dimensions
in
    GetDimensions(PngBinary)
```

## Pattern 7: Custom Binary Format Parsing

### Fixed-Width Binary Records

```m
let
    // Parse a custom binary format with fixed-length fields
    ParseFixedWidth = (binary as binary) as table =>
        let
            RecordLength = 32,  // Each record is 32 bytes
            
            // Calculate number of records
            TotalBytes = Binary.Length(binary),
            RecordCount = Number.IntegerDivide(TotalBytes, RecordLength),
            
            // Function to parse one record
            ParseRecord = (index as number) as record =>
                let
                    offset = index * RecordLength,
                    record = Binary.Range(binary, offset, RecordLength),
                    bytes = Binary.ToList(record),
                    
                    // Parse fields based on position
                    // Field 1: ID (4 bytes, integer)
                    id = bytes{0} + bytes{1} * 256 + bytes{2} * 65536 + bytes{3} * 16777216,
                    
                    // Field 2: Name (20 bytes, string)
                    nameBytes = List.Range(bytes, 4, 20),
                    name = Binary.ToText(
                        Binary.FromList(List.Select(nameBytes, each _ <> 0)),
                        TextEncoding.Ascii
                    ),
                    
                    // Field 3: Value (4 bytes, float)
                    // Simplified - would need proper float conversion
                    valueBytes = List.Range(bytes, 24, 4),
                    value = valueBytes{0} + valueBytes{1} / 256,
                    
                    // Field 4: Status (1 byte, boolean)
                    status = bytes{28} <> 0,
                    
                    // Field 5: Reserved (3 bytes)
                    reserved = List.Range(bytes, 29, 3)
                in
                    [
                        ID = id,
                        Name = name,
                        Value = value,
                        Status = status
                    ],
            
            // Parse all records
            Records = List.Transform({0..RecordCount-1}, each ParseRecord(_)),
            Result = Table.FromRecords(Records)
        in
            Result,
    
    // Usage
    BinaryContent = File.Contents("C:\Data\custom.dat"),
    ParsedData = ParseFixedWidth(BinaryContent)
in
    ParsedData
```

### Variable-Length Records with Delimiters

```m
let
    ParseDelimitedBinary = (binary as binary, delimiter as number) as table =>
        let
            bytes = Binary.ToList(binary),
            
            // Split by delimiter
            SplitByDelimiter = (byteList as list) as list =>
                let
                    positions = List.PositionOf(byteList, delimiter, Occurrence.All),
                    
                    // Add start and end positions
                    splits = List.Accumulate(
                        positions,
                        [Start = 0, Records = {}],
                        (state, pos) =>
                            let
                                record = List.Range(byteList, state[Start], pos - state[Start]),
                                newState = [
                                    Start = pos + 1,
                                    Records = state[Records] & {record}
                                ]
                            in
                                newState
                    ),
                    
                    // Add last record
                    finalRecords = splits[Records] & 
                        {List.Range(byteList, splits[Start], List.Count(byteList) - splits[Start])}
                in
                    finalRecords,
            
            Records = SplitByDelimiter(bytes),
            
            // Convert each record to text
            RecordToText = (record as list) as text =>
                Binary.ToText(Binary.FromList(record), TextEncoding.Ascii),
            
            Texts = List.Transform(Records, RecordToText),
            
            // Create table
            Result = Table.FromColumns(
                {Texts},
                {"Data"}
            )
        in
            Result
in
    ParseDelimitedBinary
```

## Pattern 8: Combining Binary Files

### Split Large Files

```m
let
    // Split a large binary into chunks
    SplitBinary = (binary as binary, chunkSize as number) as list =>
        let
            totalSize = Binary.Length(binary),
            chunkCount = Number.RoundUp(totalSize / chunkSize),
            
            chunks = List.Transform(
                {0..chunkCount-1},
                each Binary.Range(binary, _ * chunkSize, chunkSize)
            )
        in
            chunks,
    
    // Usage
    LargeFile = File.Contents("C:\Data\largefile.dat"),
    Chunks = SplitBinary(LargeFile, 1024 * 1024)  // 1MB chunks
in
    Chunks
```

## Performance Considerations

### Binary Buffering

```m
let
    // Without buffering - may be re-evaluated multiple times
    Unbuffered = File.Contents("C:\Data\largefile.dat"),
    Size1 = Binary.Length(Unbuffered),  // File read again
    Size2 = Binary.Length(Unbuffered),  // File read again
    
    // With buffering - read once, cached in memory
    Buffered = Binary.Buffer(File.Contents("C:\Data\largefile.dat")),
    Size1b = Binary.Length(Buffered),   // Uses cached binary
    Size2b = Binary.Length(Buffered)    // Uses cached binary
in
    [UnbufferedReads = Size2, BufferedReads = Size2b]
```

### Memory Management

For large binary files, process in chunks:

```m
let
    // Process large file in chunks to avoid memory pressure
    ProcessInChunks = (filePath as text, chunkSize as number, processFunction as function) as table =>
        let
            fileStream = File.Contents(filePath),
            totalSize = Binary.Length(fileStream),
            chunkCount = Number.RoundUp(totalSize / chunkSize),
            
            ProcessChunk = (chunkIndex as number) as table =>
                let
                    offset = chunkIndex * chunkSize,
                    remaining = totalSize - offset,
                    thisChunkSize = if remaining < chunkSize then remaining else chunkSize,
                    chunk = Binary.Range(fileStream, offset, thisChunkSize),
                    processed = processFunction(chunk)
                in
                    processed,
            
            AllChunks = List.Transform(
                {0..chunkCount-1},
                each ProcessChunk(_)
            ),
            Combined = Table.Combine(AllChunks)
        in
            Combined
in
    ProcessInChunks
```

## Error Handling

```m
let
    // Safe binary processing
    SafeProcessBinary = (filePath as text) as table =>
        let
            // Try to read file
            BinaryContent = try File.Contents(filePath) otherwise null,
            
            Result = if BinaryContent = null then
                // Return error table
                #table(
                    {"File", "Error"},
                    {{filePath, "File not found or access denied"}}
                )
            else
                try
                    // Attempt to process
                    Excel.Workbook(BinaryContent, null, true)
                otherwise
                    #table(
                        {"File", "Error"},
                        {{filePath, "Invalid Excel format"}}
                    )
        in
            Result
in
    SafeProcessBinary
```

## Best Practices Summary

### Do's
✓ **Use Binary.Buffer** - Cache binaries that are reused
✓ **Process in chunks** - Handle large files efficiently
✓ **Validate file signatures** - Check file type before processing
✓ **Handle errors gracefully** - Return error tables instead of failing
✓ **Document binary formats** - Comment on offsets and field meanings
✓ **Test with small samples** - Verify parsing logic first

### Don'ts
✗ **Don't load entire files into memory unnecessarily** - Use streaming where possible
✗ **Don't assume file extensions** - Validate actual binary signatures
✗ **Don't ignore encoding** - Specify TextEncoding for text conversion
✗ **Don't use Binary.Buffer on tiny files** - Unnecessary overhead
✗ **Don't parse complex formats manually** - Use built-in functions when available

## Common Binary Signatures

| File Type | Signature (Hex) | Signature (Text) |
|-----------|-----------------|-------------------|
| **PDF** | 25 50 44 46 | %PDF |
| **ZIP** | 50 4B 03 04 | PK.. |
| **XLSX** | 50 4B 03 04 | PK.. (same as ZIP) |
| **JPEG** | FF D8 | ÿØ |
| **PNG** | 89 50 4E 47 | .PNG |
| **GIF** | 47 49 46 38 | GIF8 |
| **MP3** | 49 44 33 | ID3 |

## Summary

**Key Points for Binary File Processing:**

| Aspect | Recommendation |
|--------|----------------|
| **Reading** | Use `File.Contents`, `Web.Contents`, or SharePoint functions |
| **Buffering** | `Binary.Buffer` for reused binaries |
| **Format Parsing** | Use built-in functions when available |
| **Custom Formats** | Parse with `Binary.Range` and byte lists |
| **Large Files** | Process in chunks to manage memory |
| **Error Handling** | Try/otherwise with fallback tables |
| **Performance** | Buffer, chunk, and avoid redundant reads |

**The Golden Rule:**
> Treat binary data as raw bytes until you have a reason to interpret it. Validate file signatures before processing, buffer when reused, and always handle errors gracefully. For complex formats, use built-in functions rather than reinventing the wheel.
